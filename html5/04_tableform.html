表格基本架構

  • 一個基本的表格包含「表格(table)」、「儲存格(cell)」、「欄(column)」、「列(row)」。
  • 列：橫列；欄：直欄。
  • 標記 說明
    <table> 表格的開始與結束
    <caption> 表格標題
    <thead> 定義表頭內容
    <tbody> 定義表身內容
    <tfoot> 定義表尾內容
    <th> 增加表格標題
    <tr> 增加表格列
    <td> 增加表格欄
  • thead、tfoot與tbody是描述表格結構，適合用在長表格，三者出現順序建議為thead、tfoot、tbody
  • 設定步驟：
    1. 設定表格：<table border="1">...</table>(border屬性：顯示框線)
    2. 設定橫列數目：<tr>...</tr>
    3. 設定直欄數目：<td>...</td>

CSS的border-collapse屬性

  • 此功能是用來將表格欄位邊框合併，讓表格變得更美化，
  • border-collapse: separate：此為預設值，邊框彼此間分開。
    border-collapse: separate border-collapse: separate
  • border-collapse: collapse：邊框合併為單一邊框，建議使用。
    border-collapse: collapse border-collapse: collapse
  • border-collapse: inherit：繼承自父層的border-collapse屬性值，不建議使用。

表單

  • 表單元件：文字元件、清單元件、核取元件、按鈕元件。
  • <input>文字欄位的type屬性除了「text」之外，還有「email、tel、url」屬性，建議搭配required屬性，這樣送出表單後瀏覽器就會檢查是否有正確輸入。
  • <form>...</form>標記
    • 表單的開始與結束標記。
    • 語法：<form method="post" action="abc.asp">
    • 屬性：
      1. method
        1. 設定資料傳送方式。
        2. 設定值有post與get。
        3. get：資料會直接加在url之後，安全性較差，並且有255個字數限制。
        4. post：將資料封裝後再傳送，字串長度無限制，安全性高。
      2. action
        1. 設定資料傳送目的地，表單會與asp或php等資料庫程式配合使用。
        2. 如資料量不大，可設定傳送到e-mail，語法：<form method="post" action="mailto:abc@mail.com.tw?subject="XXX" enctype="text/plain">
        3. ?subject="XXX"：設定郵件主旨。
        4. enctype
          1. 表單傳送的編碼方式，且只有method="post"才有效。
          2. enctype="application/x-www-form-urlencoded，此為預設值，可省略不寫。
          3. enctype="multipart/form-data"，用於上傳檔案的時候。
          4. enctype="text/plain"，將表單內容傳送到e-mail，enctype的設定值必須設為"text/plain"，否則會出現亂碼。
      3. target
        • 指定提交的目的視窗。
        • 屬性值 說明
          _blank 開新視窗
          _self 目前的視窗
          _parent 上一層視窗(父視窗)
          _top 最上層視窗
          頁框名稱 直接指定視窗或頁框名稱
      4. autocomplete
        設定input元件是否使用自動完成，屬性值有on與off。
      5. novalidate
        不須驗證表單，輸入novalidate即可。
    • 表單元件如下表：

  • 表單常用屬性
    屬性 說明
    autocomplete 布林值，自動完成。
    autofocus 布林值，取得焦點。
    disabled 布林值，表單禁用元件。
    max 指定最大值(數字或日期時間)，不可以小於min屬性值。
    maxlength type屬性值為text、email、search、password、tel、url，用來限制最大長度(可輸入的字數)。
    min 指定最小值(數字或日期時間)，不可以大於max屬性值。
    minlength type屬性值為text、email、search、password、tel、url，用來限制最小長度(可輸入的字數)。
    placeholder 當元件為空值時，顯示預設提醒文字。
    readonly 布林值，設定元件不可編輯。
    required 布林值，欄位值必須填寫，且提交表單前會檢查格式。
    size type屬性值為text、email、search、password、tel、url，用來指定元件的大小(寬度)，預設值為20。

    • 語法：<input type="text" name="username" value="guest" size="10" maxlength="10" />

      文字方塊的長度，若省略此屬性，則預設值為size="20"。
    • maxlength="10"
      限制輸入文字方塊的字數，例如手機號碼10碼、郵遞區號6碼……等。

      載入網頁後，自動將游標移到此文字方塊欄位。

核取元件

  • 單選核取圓鈕radio
    • 語法：<input type="radio" name="gender" value="男" />男生
         <input type="radio" name="gender" value="女" checked />女生
      請選擇性別： 男生 女生
    • type="radio"：產生單一選擇按鈕。
    • name="gender"：此屬性值相同的元件，會被視為同一組radio元件。
    • value="女"：當表單被送出時，已選取的value值會進行下一步處理，所以此屬性的值無法從外觀看出，必須在radio元件旁加上文字，才能顯示出來讓使用者了解意思。
    • checked：預設已選取。
  • 複選核取方塊checkbox
    • 語法：<input type="checkbox" name="interest" value="羽球" checked />羽球
         <input type="checkbox" name="interest" value="桌球" />桌球
         <input type="checkbox" name="interest" value="棒球" />棒球
      請選擇興趣： 羽球 桌球 棒球
    • type="checkbox"：產生一個選取方塊。
    • name="interest"：此屬性值相同的元件，會被視為同一組checkbox元件，而同一組內可以有多個checkbox元件被選擇。
    • value="羽球"：當表單被送出時，已選取的value值會進行下一步處理，所以此屬性的值無法從外觀看出，必須在checkbox元件旁加上文字，才能顯示出來讓使用者了解意思。
    • checked：預設已選取。
