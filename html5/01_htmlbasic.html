HTML5介紹

  • HTML(HyperText Markkup Language)：超文件標記語言。
  • HTML標記不分大小寫，必須由「<」與「>」符號括住，分為圍堵標記(Container Tags)與單一標記(Single Tag)。
  • 圍堵標記是利用起始標記(Start Tag)與終止標記(End Tag)將文字圍住，終止標記前會加上一個斜線「/」。
  • 單一標記只有起始標記而沒有終止標記。
  • HTML5第一行一定要有宣告，語法為：<!DOCTYPE html>。
  • 語系與編碼：
    1. 放在<head></head>裡。
    2. <html lang="zh-Hant-TW">：網頁內容使用繁體中文。
    3. <meta charset="big-5">或<meta charset="utf-8">：網頁編碼使用big5或utf-8。
  • Viewport宣告：
    1. 為了自適應網頁(RWD)所加入的宣告。
    2. 目的是告訴瀏覽器、行動裝置的寬度及高度，頁面畫面與字體比例，使用者是否可以透過平移(Scroll)和縮放(Zoom)來瀏覽整個頁面。
    3. 語法：<meta name="viewport" content="width=device-width, user-scalable=yes, initial-scale=1.0, maximum-scale=3.0, minimum-scale=1.0">
    4. 參數：
      1. width：控制寬度，可輸入「device-width」，表示寬度隨著設備寬度自動調整。
      2. height：控制高度，可輸入「device-height」。
      3. user-scalable：是否允許使用者手動縮放，可輸入0或1，也可輸入yes或no。
      4. initial-scale： 初始縮放比例，最小0.25，最大5。
      5. maximum-scale：允許使用者縮放的最大比例，最小0.25，最大5。
      6. minimum-scale：允許使用者縮放的最小比例，最小0.25，最大5。
  • 標記名稱 語法 說明
    <html> <html>...</html> 網頁的開始與結束。
    <head> <head>...</head> 放置網頁相關資訊。
    <title> <title>...</title> 網頁標題，會顯示在瀏覽器標題列。
    <meta> <meta...> 描述網頁，有利於搜尋引擎能快速找到網站。
    <body> <body>...</body> 網頁的內容。
  • 有些標記可以加上屬性(Attributes)，來改變其在網頁呈現的方式，如果超過一個以上的屬性，則以空白隔開不同的屬性。
    <起始標記 屬性名稱1="設定值1" 屬性名稱2="設定值2"……> 以此類推。
    例如：<html lang="zh-tw">、<meta charset="utf-8" name="keywords" content="HTML, CSS, JavaScript">

HTML5結構化語法架構

<!DOCTYPE html>
<html>
  <head>
    <title>網站標題</title>
  </head>
<body>
  <header>網站主題</header>
  <nav>連結選單</nav>
  <article>
  主內容
    <section>
    章節段落
    </section>
  </article>
  <aside>側邊欄</aside>
  <footer>頁尾</footer>
</body>
</html>

標記名稱 說明
<header> 顯示網站名稱、主題或主要資訊。
<nav> 連結選單。
<aside> 側邊欄。
<main> 網頁主要內容，每個頁面只能有一個<main>。
<article> 定義主內容區。
<section> 章節或段落。
<footer> 置於頁尾，顯示版權宣告、公司聯絡資訊或作者資訊。
<figure> 指定獨立的內容，例如圖片、圖表，可以使用<figcaption>標記定義標題。
<mark> 凸顯強調的文字。
