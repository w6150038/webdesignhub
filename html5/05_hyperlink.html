超連結

  • 文字超連結：<a href="http://tw.yahoo.com" target="_blank">雅虎奇摩</a>
    顯示結果：雅虎奇摩
  • 圖片超連結：<a href="index.html"><img src="image/home.jpg" border="0" alt="回首頁" title="回首頁"></a>
    顯示結果：
  • E-mail超連結：<a href="mailto:e-mail帳號1;e-mail帳號2?subject=我的意見>...</a>
    1. 收件者有2人以上時，用分號;隔開。
    2. 郵件主旨：?subject=主旨。
    3. 郵件副本：?cc=副本e-mail帳號。
    4. 郵件密件副本：?bcc=密件副本e-mail帳號。
    5. 郵件開頭文字：?body=開頭文字。
    顯示結果：寫信給我
  • href屬性
    1. 連結外部網頁：href="http://tw.yahoo.com"
    2. 連結內部網頁：href="index.html"
    3. 連結到同一網頁指定的書籤位置：href="#公司簡介"，必須搭配name屬性名稱。
    4. 連結到其他協議(Protocol)：href="https://..."、href="ftp://.."、href="mailto:abc@mail.com"
    5. 執行JavaScript：href="javascript:alert('我是帥哥');"
  • name屬性
    1. 用來設定網頁內部連結點(錨點)，在href屬性必須用「#」來識別。
    2. <a name="公司簡介"></a>
      <a href="#公司簡介"></a>
  • target屬性
    1. target="_blank"：開新視窗。
    2. target="_parent"：連結目標會開啟在目前的視窗中，如果在框架式網頁中，則會在上一層頁框開啟網頁。
    3. target="_self"：連結目標會開啟在目前的視窗中，屬於預設值。
    4. target="_top"：連結目標會開啟在瀏覽器視窗，如果在框架式網頁中，網頁中的所有頁框也會被移除。
    5. target="框架名稱"：連結目標會開啟在有指定名稱的框架中，框架名稱必須事先由框架標記所命名。
