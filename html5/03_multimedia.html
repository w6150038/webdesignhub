嵌入圖片

  • 嵌入圖片標記為<img>，屬於單一標記。
  • <img src="images/photo1.jpg" width="300" height="300" alt="這是圖片" />
  • 屬性名稱 設定值 說明
    src 圖片位置 圖片的路徑及檔名
    alt 說明文字 圖片無法顯示時的描述文字
    title 說明文字 滑鼠移到圖片時顯示的文字
    width 寬度 以像素(pixels)為單位
    height 高度 以像素(pixels)為單位
  • JPEG檔是屬於破壞性的壓縮，應一次性調整好，避免重復編輯修改。

<audio>標記

  • 音樂格式：mp3、wav、ogg。
  • 單一檔案格式語法：<audio src="music.mp3" type="audio/mpeg" controls></audio>
  • 多個檔案格式語法：
    <audio controls="controls">
      <source src="music.ogg" type="audio/ogg" />
      <source src="music.mp3" type="audio/mp3" />
    </audio>
  • 屬性名稱 說明
    src 檔案路徑
    type 播放類型：audio/mpeg, audio/ogg
    autoplay 自動播放
    controls 顯示播放面板
    loop 循環播放
    preload 預先載入
    width/height 播放面板寬度與高度

<video>標記

  • 影片格式：ogg(Theora編碼)、mp4及WebM(VPB編碼)。
  • 單一檔案格式語法：<video src="mutimedia/abc.mp4" controls="controls"></video>
  • 多個檔案格式語法：
    
									<video controls="controls">
										<source src="multimedia/abc.mp4" type="video/mp4" />
										<source src="multimedia/abc.ogg" type="video/ogg" />
									</video>

<iframe>標記

  • <iframe>標記屬於框架語法與圍堵標記，可嵌入YouTube影音，也能夠將要連結的網頁與元件直接內嵌在目前的網頁中。
  • 如果想讓YouTube影片自動播放，可在影片的連結後面加上「?autoplay=1」。
  • <a href="a.html" target="main">最新消息</a>
    <a href="b.html" target="main">重要業務公告</a>
    <a href="c.html" target="main">通關資訊即時訊息</a>
    <iframe name="main" src="a.html" width="400" height="500" seamless></iframe>
    屬性名稱 說明
    name 內置框架名稱，target屬性指定的就是name屬性所設定的名稱。
    src 指定內嵌網頁的路徑，可以是相對路徑或絕對路徑URL。
    title 描述iframe框架，有助於搜尋引擎優化(SEO)。
    frameborder 設定內置框架的框線粗細。
    width 內置框架寬度，以pixels為單位。
    height 內置框架高度，以pixels為單位。
    allow 指定權限政策(Permissions Policy)，可以開啟或禁止瀏覽器屬性和API，確保iframe的安全和隱私。
    譬如預設情況下，跨域框架是禁用地理定位，如果要開啟地理定位，語法為：
    <iframe src="page.html" allow="geolocation"><iframe>
    allowfullscreen 允許使用全螢幕模式。
    seamless 隱藏邊框及捲軸。
  • 超連結裡的target連結到名稱為main的框架，在<iframe>裡面的name屬性要指定target的名稱，src屬性則是預設連結網頁的檔名。
