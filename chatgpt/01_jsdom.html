點擊變色按鈕

  1. 建立一個 HTML 按鈕
  2. 用JavaScript讓它每次點擊時，變換不同顏色

<button id="btn_chgcolor" class="btn_chgcolor">點擊變換顏色</button>
<script>
 let a = document.getElementById('btn_chgcolor');
 const colors = [
  { background: 'red', color: 'yellow' },
  { background: 'yellow', color: 'blue' },
  { background: 'green', color: 'white' }
 ];
 function changeColor() {
  const randomIndex = Math.floor(Math.random() * colors.length);
  const newColor = colors[randomIndex];
  this.style.cssText = `background-color: ${newColor.background}; color: ${newColor.color}`;
 }
 a.addEventListener('click', changeColor);
</script>

程式說明

const colors = [
 { background: 'red', color: 'yellow' },
 { background: 'yellow', color: 'blue' },
 { background: 'green', color: 'white' }
];

這一行定義了一個 colors 陣列，其中包含了三個顏色物件。每個物件有兩個屬性： background：背景顏色。 color：文字顏色。 這樣我們就有三個顏色組合可以隨機選擇。

function changeColor() {
 const randomIndex = Math.floor(Math.random() * colors.length);
 const newColor = colors[randomIndex];
 this.style.cssText = `background-color: ${newColor.background}; color: ${newColor.color}`;
}

這個函式的作用是隨機選擇一組顏色，然後把這個顏色設定到按鈕上。

const randomIndex = Math.floor(Math.random() * colors.length);

  1. Math.random()：
    • 這個函式會回傳一個介於0（包含）到1（不包含）之間的隨機小數，例如：0.25、0.78、0.99 等。
  2. Math.random() * colors.length：
    • colors.length 代表陣列的長度，在這裡 colors.length 是 3（因為我們有三組顏色）。
    • Math.random() * 3 會產生 0 ~ 2.9999 之間的數字，像 1.45、0.89、2.73。
  3. Math.floor(...)：
    • Math.floor() 會向下取整，確保我們得到的是 0, 1 或 2（這些是 colors 陣列的索引）。
    • 例如：
      Math.floor(1.45) → 1
      Math.floor(2.73) → 2
      Math.floor(0.89) → 0
  4. 這樣，我們就能確保 randomIndex 是 0、1 或 2，對應 colors 陣列的索引。

const newColor = colors[randomIndex];

  1. 這一行是從 colors 陣列中取出隨機的顏色組合。
  2. randomIndex 可能是 0、1 或 2，所以 colors[randomIndex] 會回傳對應的物件：
    如果 randomIndex === 0，則 newColor = { background: 'red', color: 'yellow' }
    如果 randomIndex === 1，則 newColor = { background: 'yellow', color: 'blue' }
    如果 randomIndex === 2，則 newColor = { background: 'green', color: 'white' }
  3. 所以 newColor 變數會儲存這次隨機選到的顏色組合。

this.style.cssText = `background-color: ${newColor.background}; color: ${newColor.color}`;

  1. 這一行是改變按鈕的顏色，讓它變成 newColor 指定的樣式。
  2. this 代表被點擊的按鈕（因為我們用 addEventListener 綁定函式）。
  3. style.cssText 允許一次設定多個 CSS 樣式。

整體函式流程：

  1. 產生 randomIndex，確保它是 0, 1 或 2。
  2. 用這個 randomIndex 從 colors 陣列中選出一組顏色組合。
  3. 把這組顏色套用到按鈕上，改變 background-color 和 color。

完整程式流程，假設現在點擊一次按鈕：

  1. Math.random() 產生 0.78
  2. Math.floor(0.78 * 3) = Math.floor(2.34) = 2
  3. randomIndex = 2
  4. newColor = colors[2] = { background: 'green', color: 'white' }
  5. this.style.cssText = "background-color: green; color: white";
  6. 按鈕變成綠底白字
顏色可能會不同，因為 randomIndex 每次都會變化。
