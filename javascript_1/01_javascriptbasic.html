Javascript語法架構

  • <script type="text/javascript">
     Javascript語法
    </script>

  • 放在<head>標記裡：適用於一開始顯示網頁時就執行程式。
  • 放在<body>標記裡：依照網頁載入順序執行程式。

物件(Object)

  • 標準物件模型(Document Object Model-DOM)：以window為底層，包含了窗框(frame)、文件(document)等，文件裡包含圖片(image)、表單(form)、按鈕(botton)等物件。
  • 透過id、name屬性，或forms[]、images[]等物件集合就能取得物件，並且使用各自的屬性。
  • 例如：在網頁裡顯示「歡迎光臨」文字，網頁本身的物件是document，屬於window下層，語法為：
    window.document.write("歡迎光臨")，因為JavaScript程式碼與物件在同一頁，所以window可省略不寫：
    document.write("歡迎光臨")

屬性(Property)

  • 是用來設定或取得物件的屬性內容。
  • 語法：物件名稱.屬性="設定值"
    例如：document.bgcolor="yellow"

函式(Function)

  • 是先寫好的一段程式碼，可以被不同的物件、事件重複呼叫執行。
  • 使用函式必須知道定義函式的方法與所輸入的引數，及傳回何種結果。
  • 操作步驟：
    1.定義函式。
    2.呼叫函式。
  • 語法：
    function 函式名稱(參數) {
     JavaScript敘述
     ...
     ...
     return(回傳值) //有回傳值的時候才需要
    }
  • 範例1：

    <input type="button" value="呼叫alert函式" onclick="alert('哈囉');">

    說明：
    上面語法是以事件(Event)來呼叫函式，當事件發生時，函式就會被呼叫並執行，onclick是「按一下」事件，所以當使用者按一下按鈕，就會呼叫函式。
  • 範例2

    <script type="text/javascript">
     //建立add函式，並建立a, b兩個參數。
      function add(a, b) {
      //回傳a + b。
      return a + b;
     }
     //宣告sum變數，a = 5和b = 3的引數。
     let sum = add(5, 3);
     document.write(`計算結果：${sum}`);
    </script>

    按下面的連結會跳出a = 2, b = 7, a + b = 9 的訊息框：

    <script>
     function sum_1(a, b) {
      let a1 = 2; b1 = 7;
      return `a = ${a1}, b = ${b1}, a + b = ${a1 + b1}`;
     }
    </script>

    < a href="#" onclick="alert(sum_1())">按一下計算></a>

    按一下計算

    也可以用設定按鈕的方式：

    <button value="按一下計算" onclick="alert(sum_1())">按一下計算</button>

事件(Event)

  • 「事件」是指使用者的動作或系統所引發的訊號。
  • 用特定的程式來處理事件，稱為「事件處理(Event Handling)」。
  • 負責處理事件的程式稱為「事件處理程序(Event Handler)」
  • 事件處理程序(Event Handler)通常與物件(Object)相關，不同的物件會支援不同的事件處理程序。

  • 事件處理程序
    onClick 滑鼠點選物件時
    onMouseOver 滑鼠經過物件時
    onMouseOut 滑鼠離開物件時
    onLoad 網頁載入時
    onUnLoad 離開網頁時
    onError 載入發生錯誤時
    onAbort 停止載入影像時
    onFocus 滑鼠或表單元件取得焦點時
    onBlur 滑鼠或表單元件失去焦點時
    onSelect 選取表單元件內容時
    onChange 改變欄位的資料時
    onReset 重設表單時
    onSubmit 送出表單時

addEventListener()函式註冊事件

  • 當網頁程式碼較多時，想要處理表單元件，可以使用addEventListener()函式註冊事件處理函式。例如，在名稱為btn的按鈕按一下時呼叫sum()函數，語法為：

    btn.addEventListener("click",sum);

  • 如果要在按鈕上呼叫多個函式，只要多加幾行addEventListener()函式即可。
  • addEventListener()函式註冊可以在網頁一載入時就執行，只要將函式指定在window的onLoad事件即可。

運算符號

比較運算子

用來判斷兩邊的運算元是否相等、大於或小於。

比較運算子 說明 範例
> 左邊值大於右邊值則成立 a > b
< 左邊值小於右邊值則成立 a < b
== 兩邊相等則成立 a == b
!= 兩邊不相等則成立 a != b
>= 左邊值大於或等於右邊值則成立 a >= b
<= 左邊值小於或等於右邊值則成立 a <= b

算術運算子

算術運算子 說明 範例
+ 加 a + b
- 減 a - b
* 乘 a * b
/ 除 a / b
% 取餘數 a % b
-- 遞減 a--(先傳回a，然後a減1)
--a(先減1，然後傳回a)
++ 遞加 a++(先傳回a，然後a加1)
++a(先加1，然後傳回a)

邏輯運算子

邏輯運算子 說明 範例
&& AND運算(左右兩邊都成立時才傳回真(True)) a&&b
|| OR運算(只要左右兩邊有一邊成立時才傳回真(True)) a||b

指派運算子

單一指派運算子：將資料值指派給變數。例如：a = 5; //a等於5
複合指派運算子：與其它的運算子結合。
例如：
a += 5; //a等於a + 5
a -= 5; //a等於a - 5

字串

「+」號可用來連接兩個字串。
例如：a = "abc" + "def"; //a等於abcdef

運算式如果有數字也有字串，只要其中值是字串，其後面的值也會轉換成字串。
例如：
var a = 3 + 10 + "abc"; //a等於13abc
var b = "abc" + 3 + 10; //a等於abc310

if...else條件式

語法：
if(條件判斷式) {
  條件成立，則執行這裡面的程式敘述。
} else {
  條件不成立，則執行這裡面的程式敘述。
}
範例：
if(a >= b) {
  如果a大於等於b，則執行這裡面的程式敘述。
} else {
  如果a不是大於等於b，則執行這裡面的程式敘述。
}

如果條件不成立時不執行任何程式敘述，可以省略else語法。

如果條件判斷式不只一個，則可以在else語法後面再加上if條件式。
範例：
if(a == b){
  如果a等於b，則執行這裡面的程式敘述。
}else if(a > b){
  否則如果a大於b，則執行這裡面的程式敘述。
}else if(a < b){
  否則如果a小於b，則執行這裡面的程式敘述。
}

for迴圈

當程式需要重複執行時，可利用for迴圈來執行。

語法：
for(起始值; 條件判斷式; 增減值) {
  條件成立時，則執行這裡面的程式敘述。
}
範例：計算1到100的總和

<script>
 let n1 = 0;
 for (let m1 = 1; m1 <= 100; m1 ++) {
  if (m1 === 100) {
   document.write(`${m1} = `);
  } else {
   document.write(`${m1} + `);
  }
  n1 += m1;
 }
 document.write(n1);
</script>

用for迴圈計算出1+3+5+...+15的總和。

<button id="btn7" onclick="value1()">計算1+3+5+...+15的總和</button>
<script>
 function value1() {
  var x = 0; str = "";
  for (i = 1; i <= 15; i += 2) {
   if (i == 1) {
    //str = "1"
    str = i;
   } else {
    //str = "1 + 3", str = "1 + 3 + 5", str = "1 + 3+ 5 + 7", str = "1 + 3 + 3 + 7", str = "1 + 3 + 5 + 7 + 9", str = "1 + 3 + 5 + 7 + 9 + 11", str ="1 + 3 + 5 + 7 + 9 + 11 + 13", str = "1 + 3 + 5 + 7 + 9 + 11 + 13 + 15"
    str += "+" + i;
   }
   //x = 0 + 1, x = 1 + 3, x = 4 + 5, x = 9 + 7, x = 16 + 9, x = 25 + 11, x = 36 + 13, x = 49 + 15
   x += i;
  }
 alert(`${str} = ${x}`);
 }
</script>
