Canvas線上繪圖

Canvas語法：
<canvas id="id名稱" width="寬度px" height="高度px"></canvas>

<canvas>標記只有width和height屬性，如果未指定寬度與高度，預設會設定寬度為300px，高度為150px。

<script>
 function draw() {
  var canvas = document.getElementById('mycanvas');
  if (canvas.getContext)() {
    var cts = canvas.getContext('2d');
    cts.fillstyle = '#ff0000';
    cts.fillrect(0,0,150,75);
    cts.fillstyle = 'rgba(0,0,200,0.5)';
    cts.fillrect (0,0,150,75);
  }
 }
</script>
