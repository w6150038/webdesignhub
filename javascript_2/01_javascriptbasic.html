JavaScript基本概念

  • 具有易學、快速、功能強大的特點。
  • 是一種直譯式(Interpret)的描述語言。
  • 具有誇平台、物件導向、輕量的特性，通常會與其他應用程式搭配使用。
  • 除了支援Web前端應用，JavaScript還支援JSON及XML技術，能夠快速取得後端資料庫及雲端資料，達到非同步資料傳輸。
  • 瀏覽器接收到網頁檔案時會將程式碼交給渲染引擎(Render Engine)處理，HTML碼解析並建構出DOM樹狀結構(文件物件模型，Document Object Model)，CSS碼則解析並建構CSSOM樹狀結構(CSS物件模型，CSS Object Model)。
  • 常見的JS引擎：Google Chrome的V8、Apple Safari的Nitro、Microsoft的Chakra以及Mozilla Firefox的TraceMonkey。
這是一段簡單的程式碼，顯示5 + 7 = 12。

<script>
 document.write(`5 + 7 = ${5 + 7}`);
</script>

在button標記裡呼叫onclick事件處理程序，可省略<script>標記。

<button type="button" onclick="document.getElementById(`showtime`).innerHTML = Date()"顯示現在時間</button>

<script>標記

  • <script type="text/x-scheme"></script>：使用Scheme語言。
  • <script type="text/vbscript"></script>：使用VBscript語言。
  • <script type="text/javascript"></script>：使用JavaScript語言。
  • HTML5預設是JavaScript語言，所以type屬性可以省略。

瀏覽器主控台(console)

  • 用來做資料的驗證與除錯，Google Chrome按F12可呼叫出主控台。
  • 調整console字型大小：
    ctrl + 「+」：放大。
    ctrl + 「-」：縮小。
    ctrl + 「0」：還原。

console.log()

  • <script>
      console.log(`console顯示5+7=`, (5+7));
    </script>

  • console.log()裡的文字包含一個字串(`console顯示5+7=`)與一個運算式(5+7)，以逗號,分開，console面板會以不同顏色區分。

  • <script>
      console.log(`console顯示5+7=` + (5+7));
    </script>

  • 也可以寫成`console顯示5+7 + (5+7)`，輸出成字串，加號+代表連接的意思，可接字串或運算式。
  • 按F12可看見結果，第1個顯示的是「字串+運算式」，運算式的答案會以「紫色」顯示；第2個顯示的是「字串」

console.assert()

  • 是一種邏輯判斷式，結果只有「真(True)」與假「False」，如果是「假」，則輸出「錯誤訊息」。
  • 語法：console.assert(assertion, "錯誤訊息")

  • <script>
      x = 5;
      console.assert(x > 10, "這是console.assert顯示的錯誤訊息，x沒有大於10");
    </script>

console.error()

  • 將錯誤訊息輸出到主控台，括弧內是放置要顯示的訊息，可以是字串或物件。
  • 語法：console.error(錯誤訊息)

console.warn()

  • 將警告訊息輸出到主控台，括弧內是放置要顯示的訊息，可以是字串或物件。
  • 語法：console.warn(警告訊息)

console.clear()

  • 清除主控台上的訊息，執行後主控台會輸出「Console was cleared」的訊息。
  • 也可以直接在主控台面版的任一輸出行按右鍵，點選「Clear console」來清除訊息。
  • 語法：console.clear()

console.count()

  • 顯示調用次數，括弧內可以放置要辯識的標籤，不加標籤則以預設值顯示。
  • 語法：console.count(label)

  • <script>
      console.count(); //第1次調用
      console.count("A"); //第1次調用
      console.count("A"); //第2次調用
      console.count("B"); //第1次調用
      console.count("A"); //第3次調用
      console.count(); //第2次調用
    </script>

time()、timeEnd()

  • 計算程式執行的時間長度，單位：亳秒(ms)，如果有多個程式要計時，可以在括弧內加上標籤，開始計時與結束計時的標籤名稱要一樣。
  • 語法：console.time(label) //開始計時
       console.timeEnd(label) //結束計時

    <script>
     console.time(`for loop`); //開始計時
     let x = 0;
     for (i = 1; i <= 100; i++) {
      x += i;
     }
     console.log(`x = ${x}`);
     console.timeEnd(`for loop`) //結束計時
    </script>

JavaScript基本架構

  • JavaScript程式是由一行行的程式敘述(statement)組成，包含變數、運算式、運算子、關鍵字與備註文字等等。
  • 每一行程式敘述結束後須加上分號;。
  • 當遇到區塊結構時，會使用大括號「{}」來包圍程式敘述，就不需要再加分號。例如：

    var func = () => {
     var x, y;
     x = 2;
     y = 3;
     document.write(x + y);
    }

document.write()

  • document.write是屬於document物件的寫入方法，可以在文件流(Document Stream)中寫入字串，完整的document.write()應該要包含open()與close()，例如：
    //打開文件流
    document.open();
    //寫入字串
    document.write(寫入的字串);
    //關閉文件流
    document.close();
  • 將「<h3>Hello World</h3>」字串寫入HTML文件裡：

    <h1>Hello</h1>
    <h2>World</h2>
    <script>
     document.write(`<h3>Hello World</h3>`)
    </script>

    Hello


    World


  • 如果將程式修改為等到網頁完全載入window.onload後再去執行document.write()，原本的HTML DOM就會被清空，例如：

    <h1>Hello<h1>
    <h2>World</h2>
    <script>
     window.onload = (event) => {
      document.write(`<h3>Hello World</h3>`);
     }
    </script>
