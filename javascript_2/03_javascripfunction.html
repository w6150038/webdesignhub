函式(Function)

  • 也稱為函數，是一組定義好的程式敘述，當主程式需要使用函式內定義的程式時，只要叫呼該函式，就可以執行。
  • 可重複呼叫使用，簡化程式流程。
  • 程式除錯容易。
  • 便於分工合作完成程式。
  • 函式必須先行定義，定義好的函式並不會自動執行，只有在程式中呼叫該函式名稱之後，才會執行該函式。

定義函式

  • <script>
     function 函式名稱() {
      程式敘述;
      //如果需要函式回傳值給主程式，可用return敘述來傳回資料，無回傳值的話可省略。
      renturn 回傳值;
     }
    </script>

呼叫函式

  • 函式名稱();

  • <script>
      //定義myJob函式
     function myJob() {
      document.write("我的工作是網頁程式設計！");
     }
     //呼叫myJob函式
     myJob();
    </script>

函式參數

  • 函式可以將參數(parameter)傳入函式裡，成為函式裡的變數。
  • 函式參數只會存活在函式裡，函式執行完成後也會跟著結束。
  • 參數與參數之間必須以逗號,隔開，呼叫函式的引數(Argument)數量最好要與函式所定義的參數數量相等。
  • JavaScript呼叫函式時，並不會對引數數量做檢查，只會從左到右將引數與參數配對，當引數少於參數時， 少的會傳回undefined，當引數多於參數時，多的會被忽略。
  • arguments物件：可以取得引數陣列，利用arguments.length屬性可取得函式傳入引數的數量。

  • <script>
     function 函式名稱(參數1, 參數2..., 參數n)(...) {
      程式敘述;
     }
     函式名稱 (引數1, 引數2,...,引數n);
    </script>

    <script>
     //定義myScore1函式，並設定3個參數。
     function myScore1(stu_Name, stu_Math, stu_English) {
      document.write(`引數數量：` + arguments.length);
      document.write(`學生姓名：${stu_Name}，數學成績：${stu_Math}，英文成績：${stu_English}`);
     }
     myScore1(`Daniel`, `90`, `100`);
     //當引數少於參數時，少的會回undefined。
     myScore1(`Jennifer`, `60`);
     //當引數多於參數時，多的會被忽略。
     myScore1(`John`, `70`, `90`, `92`);
    </script>

  • 當引數少於參數時，少的會回undefined，此時可先指定參數預設值，避免傳回undefined。

    <script>
     //定義myScore2函式，並設定3個參數預設值。
     function myScore2(stu_Name = "", stu_Math = 0, stu_English = 0) {
      document.write(`引數數量：` + arguments.length);
      document.write(`學生姓名：${stu_Name}，數學成績：${stu_Math}，英文成績：${stu_English}`);
     }
     myScore2(`Daniel`, `90`, `100`);
     //當引數少於參數時，少的會傳回0。
     myScore2(`Jennifer`, `60`);
     //當引數多於參數時，多的會被忽略。
     myScore2(`John`, `70`, `90`, `92`);
    </script>

  • 也可以利用typeof來判斷資料型態。

    <script>
     function myScore3(stu_Name, stu_Math, stu_English) {
      if (typeof stu_English === `undefined`) {
       stu_English = 0;
      }
      document.write(`引數數量：` + arguments.length);
      document.write(`學生姓名：${stu_Name}，數學成績：${stu_Math}，英文成績：${stu_English}`);
     }
     myScore3(`Daniel`, `90`, `100`);
     //當引數少於參數時，少的會傳回0。
     myScore3(`Jennifer`, `60`);
     //當引數多於參數時，多的會被忽略。
     myScore3(`John`, `70`, `90`, `92`);
    </script>

  • 空字車(" "、' '或` `)、0、-0、null、NaN、undefined轉換為boolean都是false。

函式回傳值

  • 如果希望能取得函式執行處理之後的結果，可以利用return敘述回傳。
  • 如果只有輸入return，而沒有回傳值，表示只終止函式執行，結果會回傳undefined。
  • 語法：return value(回傳值);

  • <script>
     //建立add函式，並建立a, b兩個參數。
     function add(a, b) {
      //回傳a + b。
      return a + b;
     }
     //宣告sum變數，a = 5和b = 3的引數。
     let sum = add(5, 3);
     document.write(`計算結果：${sum}`);
    </script>

  • <script>
     //建立checkage變數，並建立age參數，起始值0。
     function checkage(age = 0) {
      document.write(`您的年齡：${age}歲，`);
      if (age < 18) {
       return `未成年`;
      }
      return `成年`;
     }
     document.write(checkage(17));
     document.write(checkage(20));
    </script>

  • renturn可以返回任何資料型態的值，如數字、字串、布林值、物件、陣列和另一個函式。

    <script>
     //建立getUser函式。
     function getUser() {
      //在return裡建立各資料型態的值，公式是：名稱: 資料,，例如：name: `Daniel`,。
      //return包起來的程式要用大括弧{}。
      return {
       id: `008285`,
       name: `Daniel`,
       age: 25,
      }
     }
     //宣告useer變數，取得getUser函式。
     let user = getUser();
     document.write(`員工編號：${user.id}`);
     document.write(`員工姓名：${user.name}`);
     document.write(`員工年齡：${user.age}`);
    </script>

  • 高階返回函式

    <script>
     function createNum(x) {
      return function(y) {
       document.write(`x = ${x}; y = ${y}`);
       document.write(`${x} * ${y} = `);
       return x * y;
      }
     }
     let double = createNum(3);
     document.write(double(9));
    </script>

  • <script>
     //定義myAvg函式，並設定3個參數與起始值。
     function myAvg(stu_Name = "", stu_Math = 0, stu_English = 0) {
      document.write(`學生姓名：${stu_Name} 數學成績：${stu_Math} 英文成績：${stu_English}`);
      //建立stu_Avg變數並計算數學與英文的平均分數。
      let stu_Avg = (stu_Math + stu_English) / 2 ;
      //回傳stu_Avg計算結果。
      return stu_Avg;
     }
     //方法1：直接用documwnt.write取得myAvg函式回傳值。
     document.write(`平均成績：` + myAvg(`John`, 70, 80));
     //方法2：宣告avg變數取得myAvg函式回傳值。
     let avg = myAvg(`Daniel`, 90, 100);
     document.write(`平均成績：` + avg);
    </script>

函式多重用法

  • JavaScript的函式屬於一級函式(First-class Function)。
  • 一級函式特性：
    1. 可以指定給變數。
    2. 可以當作引數傳給函式使用。
    3. 可以作為函式的回傳值。

函式宣告式(Function Declaration，簡稱FD)

  • 函式宣告在執行階段之前就會被建立，因此具有提升Hoisting的特性。
  • 整個程式同一個作用域(Scope)內都可以呼叫這個函式。
  • 呼叫函式方法放在函式定義之前或之後都可以。

  • <script>
     myfunc(30, 50);
     function myfunc(a, b) {
      document.write(`a = ${a}, b = ${b});
     }
     myfunc(100, 200);
    </script>

遞迴呼叫(Recursive Calls)

  • 遞回函式(Recursive Function)簡單來說就是在一個函式當中再去呼叫它自己，其中一個實際的範例就是階層的計算(Factorial)。
  • 遞回函式通常需要有兩項條件，一項是重複執行它自己，另一項則是跳出執行的出口，避免造成無窮迴圈。
  • 能夠使用遞回函式，是因為函式堆疊（Stack）的特性，當函式呼叫另一個函式時，需等候裡面的函式執行完，才會繼續回來執行自己的函式內容，應用到堆疊（Stack）資料結構後進先出（LIFO）的概念。

  • <script>
     function factorial_1() {
      for (i = 5; i >= 1; i--) {
       if (i === 1) {
        document.write(`${i} = `);
       } else {
        document.write(`${i} * `);
       }
      }
     }
     function factorial_2(num) {
      if (num === 1) {
       return num;
      } else {
       return num * factorial_2(num - 1);
      }
     }
     document.write(`5! = `) + factorial_1();
     document.write(factorial_2(5));
    </script>

箭頭函式(開頭沒有function)

  • 單行寫法(不用加return)：

    <script>
     const num1 = (a, b) => a * b;
     document.write(`計算結果：${num1(2, 3)}`);
    </script>

  • 多行寫法(需要加return)

    <script>
     const num2 = (a, b) => {
      let r = a * b;
      return `a = ${a}, b = ${b}, a * b = ${r}`;
     }
     document.write(num2(5, 9));
    </script>

函式表達式

  • 若將函式指派給變數，這種定義方式稱之為「函式表達式(Function Expressions，簡稱FE)」。
  • 函式初始化之後就不會再變動的，所以變數通常使用const宣告。
  • 語法：
    const 變數 = function(參數1, 參數2,...,參數n) {
     程式敘述;
     return 回傳值;
    }

匿名函式

  • 是指沒有名稱的函式。

  • const myfunc = function(a, b) {
     return a * b;
    }
    myfunc(3, 5);

  • 匿名函式可以使用箭頭函式來表示。語法：
    const 變數 = (參數1, 參數2,...,參數n) => {
     程式敘述;
     return 回傳值;
    }

  • <script>
     const myfunc1 = (a, b) => {
      let c = a * b;
      return `a = ${a}; b = ${b}; a * b = ${c}`;
     }
     document.write(myfunc1(10, 20));
    </script>

  • 如果函式區塊內只有一行程式碼，則箭頭函式就可以移除大括弧，並內建return回傳值。

    <script>
     const myfunc2 = (a, b) => `a = ${a}; b = ${b} a + b = ${a + b}`;
     document.write(myfunc2(10, 20));
    </script>

具名函式

  • 是指function後面加上名稱。
  • 只能在函式內部呼叫，在函式之外是不存在的。

  • const myfunc = function add(a, b) {
     return a * b;
    }
    myfunc(3, 5);

  • 五階乘式用具名函式寫法：

    <script>
     const factorial_3 = function fac_1() {
      for (i = 5; i >= 1; i--) {
       if (i === 1) {
        document.write(`${i} = `);
       } else {
        document.write(`${i} * `);
       }
      }
     }
     const factorial_4 = function fac_2(num) {
      if (num === 1) {
       return num;
      } else {
      return num * fac_2(num - 1);
      }
     }
     document.write("5! = ") + factorial_3();
     document.write(factorial_4(5));
    </script>

  • <script>
     //建立checkflag函式，參數為flag。
     function checkflag(flag) {
      //宣告myfunc變數，並設定為空值。
      let myfunc = null;
      //如果參數flag為true或false，
      if (flag) {
       //則建立myfunc函式裡a, b兩個引數，並且相加；
       myfunc = (a, b) => `a = ${a}, b = ${b}, a + b = ${a + b}`;
      } else {
       //否則將myfunc函式裡a, b兩個引數相乘。
       myfunc = (a, b) => `a = ${a}, b = ${b}, a * b = ${a * b}`;
      }
      document.write(myfunc(10, 20));
     }
     //flag裡的參數為true，表示它是有值，所以a, b會相加。
     checkflag(true);
     //flag裡的參數為false，表示它是空值，所以a, b會相乘。
     checkflag(false);
    </script>

物件與this關鍵字

物件

  • 函式本身也是物件，跟一般物件一樣具有「屬性(Properties)」與「方法(Method)」。
  • 在主控台查看物件的屬性與方法：console.dir 函式名稱();。

  • <script>
     function obj() {
      console.log(`hello`);
     }
     console.dir(obj);
    </script>

  • 任何事跟物都可以視為「物件」，以杯子來說，杯子是一個「物件」，杯子的外觀(容量、形狀、顏色)屬於「屬性」。 杯子的功能(裝水、開蓋等動作)，屬於物件的「方法」，「屬性」與「方法」屬於物件的「成員」。
  • 定義物件方式-1：使用大括弧{}的物件實字(Object Literal)，此方式比較常用。
    let object = {物件成員};
  • 物件是由一對大括弧包著一個以上的成員所組成，各個成員是一種name/value的結構，各成員以逗號(,)隔開，name(名稱) 與value(值)之間以冒號(:)隔開。
    //宣告object物件，並以大括弧包住物件成員。
    let object = {
     //成員1
     name1 : value1,
     //成員2
     name2 : value2,
     //成員3
     name3 : value3
    }
  • 物件成員的值的資料型態是數字、字串、陣列或函式皆可，如果成員的值是定義物件的樣式，就屬於該物件的「屬性」； 如果成員的值是定義物件要做的事情，就屬於該物件的「方法」，並且把要做的事情放在函式裡。
    //宣告car物件。
    let car = {
     //屬性color。
     color : "red",
     //屬性speed。
     speed : 100,
     //drive方法。
     drive() {
      return "車子啟動"
     }
    }

    定義好物件後，要存取屬性與方法，可以使用點記法(Dot Notation)。
    //取得car物件的color屬性值。
    car.color
    //執行car物件的drive()方法。
    car.drive()

  • 定義物件方式-2：使用function函式建立物件，再使用new關鍵字產生物件實體。
    function person() {物件成員}
    const object = new object();

  • <script>
     //宣告person變數為物件。
     let person = {
      //name資料用陣列方式。
      name : ["Daniel", "Mary"],
      address : "臺中市",
      mail : "abc@hotmail.com",
      age : 28,
      txt() {
       //用this關鍵字指定屬性。
       document.write(`您好，我是${this.name[0]}，今年${this.age}歲。`);
       document.write(`我家住在${this.address}，我的e-mail是：${this.mail}。`);
      }
     }
     //原本age是28，現改為35。
     person.age = 35;
     person.tel = "04-22221111";
     person.work = function() {
      document.write(`${this.name[1]}正在努力工作賺錢。`);
     }
     //呼叫person的txt函式。
     person.txt();
     //呼叫person的work函式。
     person.work();
    </script>

this關鍵字

  • this的指向並沒有固定哪一個目標，而是看呼叫的物件是誰。

  • <script>
     //宣告color字串變數，值為綠色。
     var color = `綠色`;
     //建立showColor函式，用this指向color變數綠色。
     function showColor() {
      return this.color;
     }
     //宣告carA物件，屬性drive呼叫showColor函式(不用再加小括弧)，且this會指向carA的屬性紅色。
     let carA = {
      color : `紅色`,
      drive : showColor,
     }
     //宣告carB物件，屬性drive呼叫showColor函式(不用再加小括弧)，且this會指向carB的屬性藍色。
     let carB = {
      color : `藍色`,
      drive : showColor,
     }
     document.write(`車輛A顏色：${carA.drive()}`);
     document.write(`車輛B顏色：${carB.drive()}`);
     document.write(`原本預設顏色：${showColor()}`);
    </script>

立即執行函式(Immediately Invoked Function Expression，簡稱IIFE)

  • 在函式表達式後面加上括號()，JS會立即執行該函式，

  • <script>
     let hellofunc = function() {
      document.write(`您好！`);
     } ();
    </script>

  • <script>
     const num3 = function(a ,b) {
      return `a = ${a}; b = ${b}; a * b = ${a * b}`
     } (10, 20);
     document.write(num3);
    </script>

  • IIFE如果不需要return回傳值，就不需要宣告變數來接收執行結果，只要使用小括弧()將函式包起來即可。

    <script>
     (function() {
      document.write("小括弧包在裡面");
     }) ();
    </script>

    <script>
     (function() {
      document.write("小括弧包在外面");
     } ());
    </script>

use strict嚴謹模式

  • <script>
     (function() {
      'use strict';
      程式敘述;
     }) ();
    </script>

  • 嚴謹模式放在整個JS程式首行，全部JS程式都會以嚴謹模式執行；如果放在函式裡的首行，那麼就只有該函式裡的敘述會套用嚴謹模式。
嚴謹模式常見的限制：
  1. 變數必須先使用var、let或const宣告才能使用。
  2. 禁止變數名稱重複。
  3. 新制的八進位值使用0o前綴(數字0 + 小寫英文字母o)，因此不允許使用舊制的八進位值。
  4. 禁止this敘述指向全域。
