JavaScript語法架構

  • <script type="text/javascript">
     JavaScript程式碼;
    </script>

  • 利用JavaScript將文字「JavaScript好簡單」顯示在<div>元件，因為瀏覽器顯示網頁會先解析 HTML、CSS、JavaScript並建立相對應的DOM樹，如果想要利用JavaScript來操控HTML文件，就必須等到 DOM樹完整渲染，否則會出現元件不存在的錯誤訊息。

    <!DOCTYPE html>
    <html>
    <head>
    <meta charset="utf-8">
    <script>
     window.onload = (event) => {
      document.getElementById("message").innerHTML = "JavaScript好簡單！";
     }
    </script>
    </head>
    <body>
     <div id="message"></div>
    </body>
    </html>

    顯示結果
  • window.onload：當網頁所有資源加載完成時會觸發load事件，因此，想要存取網頁元件時，可以加上此方法確保網頁資源全部載入完成，DOM樹可被讀取。
  • 如果只將「document.getElementById("message").innerHTML = "JavaScript好簡單！";」寫在<head>裡，會出現錯誤訊息。
  • 若將JavaScript程式寫在</html>下方，是指上面的DOM樹都已經渲染完成，就不需要使用window.onload()方法了，例如：

    <!DOCTYPE html>
    <html>
    <head>
     <meta charset="utf-8">
    </head>
    <body>
     <div id="message"></div>
    </body>
    </html>
    <script>
     document.getElementById("message").innerHTML = "JavaScript好簡單！";
    </script>

  • window的load事件也可以用「事件處理程序addEventListener()」來註冊監聽，例如：

    window.addEventListener("load", (event)) => {
     程式敘述;
    }

連結外部JS檔案

在<head>標記裡載入js檔案，語法：

 <head>
  <script src="檔案.js"></script>
 </head>

瀏覽器的Cache(快取)

  • 可加快網頁載入速度，減少伺服器負擔。
  • 若有修改網頁，但使用者快取時間還未到期時，瀏覽器只會顯示暫存的舊資料，除非使用者手動清除快取。
  • 為避免使用者瀏覽舊資料，建議可在修改CSS檔、JS檔或圖檔之後，將連結的檔名後方加上問號?以及字串(版號)，例如：
    <script src="檔案.js?20250218"></script>

  • 函數(function)註解包含功能、參數及傳回值。

    多行註解：用斜線加星號/*註解文字*/

資料型態(Data Type)

  • 資料型態：字串(String)、數字(Number)、布林(Boolean)、undefined(未定義)和Null(空值)。
  • 物件型態：物件(Object)、Synbol(符號)。

數值(Number與BigInt)

  • Number是數值型態的物件，包含整數(integer)與浮點數(float)。
  • Number.MAX_SAFE_INTEGER屬性可以取得最大安全整數值，例如：
    取一個變數為a的最大安全整數值。

    <script>
     const a = Number.MAX_SAFE_INTEGER;
     document.write(`取得結果為：${a}`);
    </script>

  • 當數值超出最大安全整數時，會發生精確度誤差的情況，這時候可以使用BigInt型態執行整數運算。
  • 要將數值定義為BigInt型態，只要在數值後面加上n，例如：9007199254740991n。

  • 以下程式得到結果因為超出最大安全整數，所以計算會有錯誤：

    <script>
     //建立一組函式bignumadd，參數為a和b，並宣告result1為整數0，result1 = a + b。
     function bignumadd(a, b) {
      let result1 = 0;
      result1 = a + b;
      return result1;
     }
     //計算bignumadd裡的a, b兩個參數的值，並將結果傳給r。
     const r = bignumadd(9007199254740991, 9007199254740996);
     document.write(`r = ${r}`);
    </script>

    以下程式才是正確的計算結果：

    <script>
     //建立一組函式bigintadd，參數為a和b，並宣告result2為整數0，result2 = a + b。
     function bigintadd(a, b) {
      let result2 = 0;
      result2 = a + b;
      return result2;
     }
     //計算bigintadd裡的a, b兩個參數的值，並將數字結尾加上n，轉換為BigInt型態，然後將結果傳給c。
     const c = bigintadd(9007199254740991n, 9007199254740996n);
     document.write(`c = ${c}`);
    </script>

  • 根據IEEE 754標準的浮點數表示小數點也會有精確度誤差的問題，必須使用toFixed()函數強制取得小數點的指定位數。
    以下程式的小數點因為是浮點數，電腦計算時只認識0和1，再將十進位轉換成二進位計算，所以會產生誤差：

    <script>
     var d = 0.58 + 0.3;
     docment.write(`d的結果為：${d}`);
    </script>

    使用toFixed()函數可強制取到小數點的指定位數。
    語法：變數.toFixed(小數點位數)，例如：

    <script>
     var e = 0.58 + 0.3;
     docment.write(`e的結果為：${e.toFixed(2)}`);
    </script>

字串(String)

  • 用一對雙引號("")、單引號('')或反引號(``)框住字元，不輸入任何字元就稱為空字串。
  • length是字串物件的屬性，用來取得字串的長度。

  • <script>
     var mystring = "Hello, World!";
     document.write(`${mystring}`);
     document.write(`「Hello, World!」的字元長度：${mystring.length}`);
    </script>

布林(boolean)

  • 布林型態的值只有true(1)與false(0)，任何值都可以被轉換成布林值。
  • false、0、空字串、NaN、null及undefined都會成為false(0)，其他的值都會成為true(1)。
  • 可以利用Boolean()函數將值轉換成布林值，但JavaScript會自己進行布林轉換，所以很少會需要用到Boolean()函數。

未定義(Undefined)

  • 是指變數沒有被宣告或是有宣告變數，但尚未指定變數的值。
  • typeof：是用來判斷變數的型態。
  • 三個等號(===)：是嚴格相等，用來比較左右兩邊是否相等。
  • 設定變數x，但未指定x的值，則會顯示undefined，另外用typeof判斷變數x的型態是否為undefined，是的話傳回true，否的話傳回false。

    <script>
     let x;
     document.write(`變數x沒有指定值，所以會傳回undefined：x = ${x}`);
     document.write(`判斷變數x是否為undefined，結果為：`);
     document.write(typeof x === `undefined`);
    </script>

空值(Null)

  • 當想要將某個變數的值清除，可以使用「變數 = null」來清除。

  • <script>
     let y = 2;
     document.write(`y = ${y}`);
     y = null;
     document.write(`y的值被清除後，得到的結果是：${y}`);
    </script>

無效的數字(NaN)

  • 如果碰到無意義的運算，例如0 / 0，也會傳回NaN。
  • isNaN()函數是用來檢查是否為無效的數字，是的話傳回true，否的話傳回false。
  • 進行運算時的運算元資料型態無法轉換為數字，就會傳回NaN；Number()函數是將物件轉換為數字，例如：

    <script>
     let i = `a`;
     j = Number(i);
     document.write(`得到的結果：j = ${j}，`);
     if (isNaN(j)) {
      document.write(`j是無效的數字。`);
     } else {
      document.write(`j不是無效的數字。`);
     }
    </script>

無限大(infinity)

  • isFinite()函數是用來檢查是否為無限大，也就是是否為有限數值。

  • <script>
     document.write(`2 / 1是否為無限大：${isFinite(2 / 1)}`);
     document.write(`2 / 0是否為無限大：${isFinite(2 / 0)}`);
     document.write(`-1 / 0是否為無限大：${isFinite(-1 / 0)}`);
     document.write(`-2 / 1是否為無限大：${isFinite(-2 / 1)}`);
    </script>

物件(Object)

物件型態種類：function(函數)、Object(物件)、Array(陣列)、Date(日期)等等， 例如：{age:'17'}(物件)、[1, 2, 3](陣列)、function a(){...}(函數)、new Date()(日期)。

符號(Symbol)

  • symbol()：內部會建立一個獨特的識別id ，兩個相同值的Symbol是不同的，且無法透過Symbol.keyFor()找到。
  • Symbol.for()：一樣都會生成新的Symbol，但兩個相同值的Symbol會是相同的識別id，且可以透過Symbol.keyFor()找到。

  • <script>
     let aa1 = Symbol(`s`);
     let aa2 = Symbol(`s`);
     let bb1 = Symbol.for(`k`);
     let bb2 = Symbol.for(`k`);
     document.write(`aa1的符號是否跟aa2相同？${aa1 === aa2}`);
     document.write(`bb1的符號是否跟bb2相同？${bb1 === bb2}`);
    </script>

變數宣告與作用範圍

  • JavaScript屬於動態語言，變數不須宣告型態，使用上更具有彈性。
  • JavaScript會在變數宣告與使用時動態配置記憶體，並具有回收記憶體的機制(Garbage Collection，簡稱GC)。
  • JavaScript的GC機制無法由程式去控制回收，而是經過一段時間後自動尋找不需要使用的物件，釋放記憶體。
  • 當變數的作用範圍結束，就不需要使用了，這時候GC就會將記憶體釋放。

宣告變數分配記憶體

  • var a = 123; //分配記憶體給數字
    var b = 'hello'; //分配記憶體給字串
    var obj = {a: 1, b: 'h1'}; //分配記憶體給物件
    var arr = [1, 'h1']; //分配記憶體給陣列
    var d = new Date() //分配記憶體給日期物件
    var x = document.createElement('div'); //分配記憶體給DOM物件
  • var和let都是用來宣告變數，差別在於變數的作用域；const是用來宣告常數。

var宣告變數

  • 使用var宣告變數包含「宣告」與「初始化」，所謂「初始化」是給變數一個初始值，可以先宣告變數後再指定初始值，也可以宣告一併初始化。
  • 宣告變數：var name;
  • 同一行可以宣告多個變數，以逗號,隔開：var name, score;
  • 宣告變數並初始化：var name = "Daniel", score = 75, flag = "true";
  • 宣告變數時不需要加上型態，JavaScript會自動轉換，例如：
    var v1;
    v1 = 123; //變數v1的內容為數值123
    v1 = "hello" //變數v1的內容為字串hello
  • JavaScript允許字串相加，當字串內容為數值時，使用加號+相連，運算結果仍為「字串」。
  • 當字串內容為數值，使用減號-、乘號*、除號/相連時，運算結果為「數值」。
  • 當宣告變數的初始值為null時，乘以任何數值的結果皆為0。

  • <script>
     //宣告x4到x6的變數為字串，字串內容為數值；x7為null
     var x4 = "5", x5 = "3", x6 = "1", x7 = null;
     ab1 = x4 + x5 + x6;
     ab2 = x4 - x5 - x6;
     ab3 = x7 * 55;
     document.write(`變數裡的字串內容為數值時，相加仍是字串：x4 + x5 + x6 = ${ab1}`);
     document.write(`變數裡的字串內容為數值時，相減則為數值：x4 - x5 - x6 = ${ab2}`);
     document.write(`變數裡的內容為null時，乘上任何數值皆為0：x7 * 55 = ${ab3}`);
    </script>

  • 沒有初始值的變數使用前必須先宣告，否則會出現ReferenceError錯誤。

var宣告變數的作用域

全域變數(Global Variables)

  • 是指存活在整個程式，任何地方都可以使用這個變數，直到整個程式執行結束。
  • 變數可以不宣告直接給初始值，例如：y1 = 5，因此省略宣告的變數都會被視為全域變數。

  • <script>
     //宣告y1的變數為2，這是全域變數。
     var y1 = 2;
     //定義cal2函式
     function cal2() {
      //因為cal2函式裡的y1及y2沒有宣告，仍然是屬於全域變數，所以y1的值變為5。
      y1 = 5, y2 = 1;
      document.write(`y1 + y2 = ${y1 + y2}`);
     }
     //執行cal2函式
     cal2();
     document.write(`y1 = ${y1}`);
    </script>

區域變數(Local Variables)

  • 只能存活在一個固定的區域範圍，例如函式(Function)裡面宣告的變數就只能夠在這個函式裡面使用，當函式執行 完成並返回後，變數就會失效。

  • <script:>
     //宣告x8的變數為2，這是全域變數。
     var x8 = 2;
     //定義cal1函式
     function cal1() {
      //宣告cal1函式裡的x8的變數為5，x9的變數為1，兩者皆為區域變數。
      var x8 = 5; x9 = 1;
      document.write(`x8 + x9 = ${x8 + x9}`);
     }
     //執行cal1函式
     cal1();
     document.write(`x8 = ${x8}`);
    </script>

let宣告變數

  • let宣告方式與var宣告相同。
  • let宣告語法的區塊作用域，在區塊內屬於區域變數，區塊以外屬於全域變數。
  • let宣告是比較嚴謹的宣告方式，同一區塊不可以重覆宣告同名變數。
  • let宣告與const宣告尚未初始化前皆不能操作，這一段時間稱為「暫時死區」(Temporal Dead Zone，簡稱TDZ)。
  • 下面程式裡，變數d3是在區塊內以let宣告，因為變數d3只存在區塊內，當第9行使用變數d3時就會出現d3未定義的錯誤訊息。

    <script;>
     var c1 = 5; c2 = 0;
     let d1 = 2; d2 = 0;
     {
      var c3 = c1 + c2;
      let d3 = d1 + d2;
     }
     console.log(`c3 = ${c3}`);
     console.log(`d3 = ${d3}`);
    </script>

const宣告常數

  • 常數不能重覆宣告，必須指定初始值，之後也不能再變更它的值。
  • 下面程式裡，第2行e1已經宣告常數，因此第4行再變更e1的值會出現錯誤。

    <script>
     const e1 = 10;
     //第2行e1已經宣告常數，因此第4行再變更e1的值會出現錯誤。
     e1 = 15;
     console.log(`e1 = ${e1}`);
    </script>

變數名稱的限制

  • 第一個字元必須是字母(大小寫皆可)或是底線_，之後的字元可以是數字、字母或底線。
  • 區分大小寫，var ABC不等於var abc。
  • 不能使用保留字。

強制轉換型態

  • JavaScript具有自動轉換資料型態(Coercion)的特性。
  • 下面程式裡，f2是字串，所以f3得到的結果是"35"，資料型態是字串。

    <script>
     let f1 = 3; f2 = "5";
     let f3 = f1 + f2;
     document.write(`f1 + f2 = ${f3}`);
     document.write(`f3的資料型態是：${typeof f3}`);
    </script>

  • 下面程式裡，k1是數字，k2卻輸入成字串，這樣真的會差很多。

    <script>
     function billing(k1, k2) {
      k3 = k1 + k2;
     }
     billing(10, "15");
     document.write(`您的費用是：${k3}元`);
     document.write(`k3的資料型態變成：${typeof k3}`)
    </script>

  • 撰寫程式時要防範強制轉換型態可能帶來的錯誤，因此上面的程式碼可以先寫好檢查傳入的引數是否為數字。

    <script>
     function billing(k4, k5) {
      if(typeof(k4, k5) === "number") {
       k6 = k4 + k5;
      } else {
       document.write(`請輸入數字！`)
      }
     }
     billing(10, 15);
     document.write(`您的費用是：${k6}元`);
     document.write(`k6的資料型態是：${typeof k6}`);
    </script>

轉換資料型態的內建函數

parseInt()：將文字轉換為整數

  • 由字串最左邊開始轉換，一直轉換到字串結束或遇到非數字字元為止，如果字串無法轉為數值，則會傳回NaN。

  • <script>
     f3 = parseInt("35");
     f4 = parseInt("55.87");
     f5 = parseInt("3天");
     f6 = parseInt("page 2");
     document.write(`f3 = ${f3}`);
     document.write(`f4 = ${f4}`);
     document.write(`f5 = ${f5}`);
     document.write(`f6 = ${f6}`);
    </script>

parseFloat()：將文字轉換為浮點數

  • <script>
     f7 = parseFloat("35.345");
     f8 = parseFloat("55.87");
     document.write(`f7 = ${f7}`);
     document.write(`f8 = ${f8}`);
    </script>

Number()：將物件或字串轉換為數值

  • 若物件或字串無法轉換為數值，則會傳回NaN。
  • Date()物件是以世界標準時間(UTC)1970年1月1日開始的亳秒數值來儲存時間，因此當使用Number()將Date()物件 轉換為數值就會得到1970年1月1日到程式執行當下的亳秒數。

  • <script>
     f9 = Number("10a");
     f10 = Number("11.5");
     f11 = Number("0x10");
     f12 = Number(true);
     f13 = Number(new Date());
     document.write(`f9 = ${f9}`);
     document.write(`f10 = ${f10}`);
     document.write(`f11 = ${f11}`);
     document.write(`f12 = ${f12}`);
     document.write(`f13 = ${f13}`);
    </script>

typeof：傳回資料型態

  • typeof: 資料
    typeof: (資料)
    以上兩種方式都可以使用。

  • <script>
     f14 = "Daniel";
     f15 = 123;
     f16 = null;
     f17 = NaN;
     document.write(`f14的資料型態是：${typeof f14}`);
     document.write(`f15的資料型態是：${typeof f15}`);
     document.write(`f16的資料型態是：${typeof f16}`);
     document.write(`f17的資料型態是：${typeof f17}`);
    </script>

  • typeof帶入任何資料都會傳回「字串」，如果是尚未宣告的變數會傳回「undefined」。

    <script>
     document.write(`傳回字串undefined：${typeof f18}`);
     document.write(`傳回undefined：${f18}`);
     var f18;
    </script>
