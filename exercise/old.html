改變按鈕顏色

  1. 建立一個 HTML 按鈕
  2. 用JavaScript讓它每次點擊時，變換不同顏色

	<button id="btn_chgcolor" class="btn_chgcolor">點擊變換顏色</button>
	<script>
		let a = document.getElementById('btn_chgcolor');
	
		// 顏色選項
		const colors = [
			{ background: 'red', color: 'yellow' },
			{ background: 'orange', color: 'red' },
			{ background: 'yellow', color: 'black' },
			{ background: 'green', color: 'white' },
			{ background: 'blue', color: 'yellow' },
			{ background: 'indigo', color: 'orange' },
			{ background: 'purple', color: 'white' }
		];
	
		// 顏色隨機變換
		function changeColor() {
			const randomIndex = Math.floor(Math.random() * colors.length);
			const newColor = colors[randomIndex];
			this.style.cssText = `background-color: ${newColor.background}; color: ${newColor.color}`;
		   }

		// 綁定事件
		a.addEventListener('click', changeColor);
	</script>

程式說明

const colors = [
 { background: 'red', color: 'yellow' },
 { background: 'orange', color: 'red' },
 { background: 'yellow', color: 'black' },
 { background: 'green', color: 'white' },
 { background: 'blue', color: 'yellow' },
 { background: 'indigo', color: 'orange' },
 { background: 'purple', color: 'white' }
];

這一行定義了一個 colors 陣列，其中包含了七個顏色物件。每個物件有兩個屬性： background：背景顏色。 color：文字顏色。 這樣我們就有七個顏色組合可以隨機選擇。

function changeColor() {
 const randomIndex = Math.floor(Math.random() * colors.length);
 const newColor = colors[randomIndex];
 this.style.cssText = `background-color: ${newColor.background}; color: ${newColor.color}`;
}

這個函式的作用是隨機選擇一組顏色，然後把這個顏色設定到按鈕上。

const randomIndex = Math.floor(Math.random() * colors.length);

  1. Math.random()：
    • 這個函式會回傳一個介於0（包含）到1（不包含）之間的隨機小數，例如：0.25、0.78、0.99 等。
  2. Math.random() * colors.length：
    • colors.length 代表陣列的長度，在這裡 colors.length 是 7（因為我們有七組顏色）。
    • Math.random() * 7 會產生 0 ~ 6.9999 之間的數字，像 1.45、0.89、6.73。
  3. Math.floor(...)：
    • Math.floor() 會向下取整，確保我們得到的是 0, 1, 2, 3, 4, 5, 或 6（這些是 colors 陣列的索引）。
    • 例如：
      Math.floor(1.45) → 1
      Math.floor(2.73) → 2
      Math.floor(6.89) → 6
  4. 這樣，我們就能確保 randomIndex 是 0、1、2、3、4、5 或 6，對應 colors 陣列的索引。

const newColor = colors[randomIndex];

  1. 這一行是從 colors 陣列中取出隨機的顏色組合。
  2. randomIndex 可能是 0、1、2、3、4、5或 6，所以 colors[randomIndex] 會回傳對應的物件：
    如果 randomIndex === 0，則 newColor = { background: 'red', color: 'yellow' }
    如果 randomIndex === 1，則 newColor = { background: 'orange', color: 'red' }
    如果 randomIndex === 2，則 newColor = { background: 'yellow', color: 'black' }
    如果 randomIndex === 3，則 newColor = { background: 'green', color: 'white' }
    如果 randomIndex === 4，則 newColor = { background: 'blue', color: 'yellow' }
    如果 randomIndex === 5，則 newColor = { background: 'indigo', color: 'orange' }
    如果 randomIndex === 6，則 newColor = { background: 'purple', color: 'white' }
  3. 所以 newColor 變數會儲存這次隨機選到的顏色組合。

this.style.cssText = `background-color: ${newColor.background}; color: ${newColor.color}`;

  1. 這一行是改變按鈕的顏色，讓它變成 newColor 指定的樣式。
  2. this 代表被點擊的按鈕（因為我們用 addEventListener 綁定函式）。
  3. style.cssText 允許一次設定多個 CSS 樣式。

整體函式流程：

  1. 產生 randomIndex，確保它是 0, 1, 2, 3, 4, 5 或 6。
  2. 用這個 randomIndex 從 colors 陣列中選出一組顏色組合。
  3. 把這組顏色套用到按鈕上，改變 background-color 和 color。

完整程式流程，假設現在點擊一次按鈕：

  1. Math.random() 產生 0.78
  2. Math.floor(0.78 * 6) = Math.floor(4.68) = 4
  3. randomIndex = 4
  4. newColor = colors[4] = { background: 'blue', color: 'yellow' }
  5. this.style.cssText = "background-color: blue; color: yellow";
  6. 按鈕變成藍底黃字
顏色可能會不同，因為 randomIndex 每次都會變化。

改變按鈕顏色(讓顏色不重複上一次的顏色)


	<button id="btn_chgcolor1" class="btn_chgcolor">點擊變換顏色</button>
	<script>
		let b = document.getElementById('btn_chgcolor1');

		const colors1 = [
			{ background: 'red', color: 'yellow' },
			{ background: 'orange', color: 'red' },
			{ background: 'yellow', color: 'black' },
			{ background: 'green', color: 'white' },
			{ background: 'blue', color: 'yellow' },
			{ background: 'indigo', color: 'orange' },
			{ background: 'purple', color: 'white' }
		];

		let lastIndex = -1;

		function changeColor1() {
			let randomIndex1;
			do {
				randomIndex1 = Math.floor(Math.random() * colors1.length);
			} while (randomIndex1 === lastIndex);
			lastIndex = randomIndex1;
			const newColor1 = colors1[randomIndex1];
			this.style.cssText = `background-color: ${newColor1.background}; color: ${newColor1.color}`;
		}

		b.addEventListener('click', changeColor1);
	</script>

程式說明

let lastIndex = -1;

  1. lastIndex變數是用來記錄上一次選擇的顏色索引。
  2. 初始值設為 -1，代表「還沒選過」。

do {
 randomIndex1 = Math.floor(Math.random() * colors1.length);
} while (randomIndex1 === lastIndex);

  1. do...while 會先執行一次隨機選擇。
  2. 如果 randomIndex1 和 lastIndex 相同，就再選一次，直到不相同為止。

lastIndex = randomIndex1;

更新lastIndex，這樣下一次點擊時，lastIndex 會記得上一次的顏色，確保新選擇的顏色不會跟上次一樣。

條件判斷(一) 👉🏻 if

  1. 如果程式需要根據條件執行不同的動作，就用 if。
  2. 只在特定條件發生時執行。
  3. 只做一次判斷，不需要重複執行。

如果按鈕是紅色，就變黃，否則變綠。


	<button id="btn_chgcolor2" class="btn_chgcolor" style="background-color: red; color: yellow;">點擊變換顏色</button>
	<script>
		let c = document.getElementById('btn_chgcolor2');

		c.addEventListener('click', function () {
			if (this.style.backgroundColor === 'red') {
				this.style.backgroundColor = 'yellow';
				this.style.color = 'black';
			} else if (this.style.backgroundColor === 'yellow') {
				this.style.backgroundColor = 'green';
				this.style.color = 'white';
			} else {
				this.style.backgroundColor = 'red';
				this.style.color = 'yellow';
			}
		});
	</script>

顯示現在時間與日期


	<div id="showTime"></div>
	<script>
		function showTime() {
			let now = new Date();
			let hours = now.getHours().toString().padStart(2, '0');
			let minutes = now.getMinutes().toString().padStart(2, '0');
			let seconds = now.getSeconds().toString().padStart(2, '0');
			document.getElementById('showTime').textContent = `現在時間：${hours}:${minutes}:${seconds}`
		}
		setInterval(showTime, 1000);
		showTime();
	</script>

綜合實作-顯示年月日、星期幾和現在時間


	<div id="showFullToday"></div>
	<script>
		function showFullToday() {
			let now = new Date();
			let year = now.getFullYear() - 1911;
			let month = (now.getMonth() + 1).toString().padStart(2, '0');1
			let day = now.getDate().toString().padStart(2, '0');
			let hours = now.getHours().toString().padStart(2, '0');
			let minutes = now.getMinutes().toString().padStart(2, '0');
			let seconds = now.getSeconds().toString().padStart(2, '0');
			let weekday = "";
			let d = now.getDay();
			switch (d) {
				case 0:
					weekday = '星期日';
					break;
				case 1:
					weekday = '星期一';
					break;
				case 2:
					weekday = '星期二';
					break;
				case 3:
					weekday = '星期三';
					break;
				case 4:
					weekday = '星期四';
					break;
				case 5:
					weekday = '星期五';
					break;
				case 6:
					weekday = '星期六';
					break;
				default:
					weekday = '發生錯誤！'
			}
			let t = document.getElementById('showFullToday');
			t.textContent =	`今天是民國 ${year} 年 ${month} 月 ${day} 日，${weekday}，現在時間：${hours}:${minutes}:${seconds}`;
		}

		setInterval(showFullToday, 1000);
	</script>

用for迴圈印出1到n的平方數


	計算 1 到 <input id="inputNum" type="number" min="1" placeholder="請輸入數字">的平方數。
	<button id="count" onclick="square()">計算</button>
	<div id="square"></div>
	<script>
		function square() {
			let n = Number(document.getElementById('inputNum').value);
			let r = ""; 
			for (let i = 1; i <= n; i++) {
				r += `${i} 的平方是 ${i * i}。
`;
			}
			document.getElementById('square').innerHTML = r;
		}
	</script>
