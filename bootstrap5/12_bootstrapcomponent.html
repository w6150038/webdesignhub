Collapse 可折疊區塊

Bootstrap 的 Collapse 元件可以折疊 (隱藏) 或展開某個區塊內容，常見用在 FAQ、收合式選單等地方。

class 名稱 說明
data-bs-toggle="collapse" 觸發折疊功能。
data-bs-target="#ID" 折疊的ID區塊。
collapse 放在要收合的區塊。
id="collapseExample" 要與 data-bs-target 對應。
card card-body 折疊區塊的內容。 

                                <button class="btn btn-primary" type="button" data-bs-toggle="collapse" data-bs-target="#collapseExample" aria-expanded="false" aria-controls="collapseExample">
                                顯示 / 隱藏
                                </button>
                                <div class="collapse mt-3" id="collapseExample">
                                    <div class="card card-body text-dark">
                                        這裡面是內容。
                                    </div>
                                </div>

Carousel 輪播

Carousel (輪播) 常用於建立圖片輪播區 (如首頁 banner )，能自動或手動切換圖片、文字或其它內容。

class 名稱 說明
carousel slide 建立滑動輪播。
carousel-fade 淡入淡出效果。
carousel-item active 輪播項目，第一張一定要有 active。
data-bs-ride="carousel" 自動輪播。
carousel-control-prev/next 左右切換按鈕。
data-bs-interval="3000" 輪播間隔時間，單位是毫秒 (3000 = 3 秒)。
data-bs-pause="hover" 滑鼠移到 Carousel 上時暫停自動輪播。
carousel-indicators 點選小圓點可切換到不同張圖片。
carousel-caption 每張圖片上顯示標題或描述文字。
img-thumbnail 縮圖樣式，帶有邊框與內距。

Tooltip & Popover 訊息小浮窗

Tooltip 和 Popover 兩者都要用 JavaScript 才能啟用。

元件名稱 觸發方式 顯示內容 適用情境
Tooltip 滑鼠碰到。 簡短文字。 顯示欄位說明、提示。
Popover 點擊後出現。 標題 + 內容。 顯示更多補充資訊或操作提示。

Tootip


                                <button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" title="Tooltip 顯示的文字" data-bs-placement="right">
                                    滑鼠移過來
                                </button>
                                <script>
                                    const tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]');
                                    const tooltipList = [...tooltipTriggerList].map(el => new bootstrap.Tooltip(el));
                                </script>

Popover


                                <button type="button" class="btn btn-danger" data-bs-toggle="popover" title="Popover 標題" data-bs-content="Popover 顯示的內容">
                                    滑鼠點擊
                                </button>
                                <script>
                                    const popoverTriggerList1 = document.querySelectorAll('[data-bs-toggle="popover"]');
                                    const popoverList1 = [...popoverTriggerList1].map(el => new bootstrap.Popover(el));
                                </script>

Popover (各別獨立樣式)


                                <button type="button" class="btn btn-success popover1" data-bs-toggle="popover" title="Popover 標題" data-bs-content="Popover 顯示的內容" data-bs-placement="auto">
                                    滑鼠點擊
                                </button>
                                <script>
                                    const popoverTriggerList2 = document.querySelectorAll('.popover1');
                                    popoverTriggerList2.forEach(el => {
                                        new bootstrap.Popover(el, {
                                            customClass: 'popover-style'
                                        });
                                    });
                                </script>

Popover (圖片與網址)


                                <button type="button" class="btn btn-success popover-html" data-bs-toggle="popover" title="Popover 標題" data-bs-placement="right">
                                    滑鼠點擊
                                </button>
                                <script>
                                    const htmlPopoverList = document.querySelectorAll('.popover-html');
                                    htmlPopoverList.forEach(el => {
                                        new bootstrap.Popover(el, {
                                            html: true,
                                            anitize: false,
                                            content: `
                                                <img src="https://picsum.photos/800/300?random=17" class="img-fluid mb-2" alt="示意圖">
                                                <br />
                                                <a href="https://getbootstrap.com/" target="_blank">開啟 Bootstrap 官網</a>
                                            `,
                                            customClass: 'custom-html-popover'
                                        });
                                    });
                                </script>

Popover (表單)


                                <button type="button" class="btn btn-info popover-form" id="formPopover" data-bs-toggle="popover" title="訂閱電子報" data-bs-placement="right">
                                    訂閱電子報
                                </button>
                                <script>
                                    const formPopover = document.getElementById('formPopover');
                                    if (formPopover) {
                                        new bootstrap.Popover(formPopover, {
                                        html: true,
                                        sanitize: false,
                                        ontent: `
                                            <form onsubmit="event.preventDefault(); alert('感謝訂閱！'); bootstrap.Popover.getInstance(document.getElementById('formPopover')).hide();">
                                                <div class="mb-2">
                                                <input type="email" class="form-control form-control-sm" placeholder="請輸入 E-mail" required>
                                                </div>
                                                <button type="submit" class="btn btn-primary btn-sm">送出</button>
                                            </form>
                                        `,
                                        customClass: 'popover-form'
                                        });
                                    };
                                </script>

在網頁任何地方點選一下，會把正在出現的 popover 關閉，需要用共用的 JavaScript 程式。


                                <script>
                                    document.addEventListener('click', function (e) {
                                        document.querySelectorAll('[data-bs-toggle="popover"]').forEach(triggerEl => {
                                            const popoverEl = document.querySelector('.popover');
                                            if (
                                                !triggerEl.contains(e.target) && (!popoverEl || !popoverEl.contains(e.target))
                                            ) {
                                            bootstrap.Popover.getInstance(triggerEl)?.hide();
                                            };
                                        });
                                    });
                                </script>

初階 (Beginner)

適合對象：剛開始接觸羽球，或只會基本揮拍的人。

特徵：

  • 握拍姿勢、站姿、揮拍姿勢還在學習中。
  • 發球、接發球基本動作尚不穩定。
  • 移動步伐不熟悉（容易卡腳步）。
  • 大部分回球靠運氣或本能。
  • 還不熟悉雙打站位、場地分工。
  • 幾乎無比賽經驗。

中階 (Intermediate)

適合對象：已經打球一段時間，基本技術已有基礎，能進行對打與比賽。

特徵：

  • 握拍與擊球姿勢較正確。
  • 能穩定發高遠球、吊球、平球，偶爾可殺球。
  • 步伐靈活，有一定反應速度。
  • 具備基本的戰術意識（如左右分工、吊球與殺球配合）。
  • 能參與練習賽、友誼賽等。

高階 (Advanced)

適合對象：打球多年，有正式比賽經驗，目標為競技層級或業餘菁英。

特徵：

  • 所有基本技術純熟，動作標準、效率高。
  • 擅長快速多變的球路（如連續殺球、反手撲球、快速平抽）。
  • 能讀懂對手動作，預測落點。
  • 體能佳、步伐快速。
  • 熟悉各種比賽策略與臨場應變。
