Collapse 可折疊區塊

Bootstrap 的 Collapse 元件可以折疊 (隱藏) 或展開某個區塊內容，常見用在 FAQ、收合式選單等地方。

class 名稱 說明
data-bs-toggle="collapse" 觸發折疊功能。
data-bs-target="#ID" 折疊的ID區塊。
collapse 放在要收合的區塊。
id="collapseExample" 要與 data-bs-target 對應。
card card-body 折疊區塊的內容。 

    <button class="btn btn-primary" type="button" data-bs-toggle="collapse" data-bs-target="#collapseExample" aria-expanded="false" aria-controls="collapseExample">
    顯示 / 隱藏
    </button>
    <div class="collapse mt-3" id="collapseExample">
        <div class="card card-body text-dark">
            這裡面是內容。
        </div>
    </div>
這裡面是內容。

Accordion 手風琴樣式

class 名稱 說明
accordion 外層容器。
accordion-item 每一組項目。
accordion-button 點擊的按鈕。
accordion-collapse 可收合的內容區塊
data-bs-parent="#accordionExample" 確保一次只展開一個項目。 

    <div class="accordion" id="accordionExample">
        <div class="accordion-item">
            <h2 class="accordion-header" id="headerOne">
                <button class="accordion-button" type="button" data-bs-toggle="collapse" data-bs-target="#collapseOne" aria-expanded="true" aria-controls="collapseOne">
                    問題 1：電玩的影片被挪用來說是烏俄戰爭的戰事，這種假訊息的手法叫做什麼？
                </button>
            </h2>
            <div class="accordion-collapse collapse show" id="collapseOne" aria-labelledby="headerOne" data-bs-parent="#accordionExample">
                <div class="accordion-body">
                    答案：移花接木。
                </div>
            </div>
        </div>
        <div class="accordion-item">
            <h2 class="accordion-header" id="headerTwo">
                <button class="accordion-button" type="button" data-bs-toggle="collapse" data-bs-target="#collapseTwo" aria-expanded="false" aria-controls="collapseTwo">
                    問題 2：手機收到一段傳言，會讓自己感覺到很生氣，應該怎麼做？
                </button>
            </h2>
            <div class="accordion-collapse collapse" id="collapseTwo" aria-labelledby="headerTwo" data-bs-parent="#accordionExample">
                <div class="accordion-body">
                    答案：先查證，再轉傳。
                </div>
            </div>
        </div>
        <div class="accordion-item">
            <h2 class="accordion-header" id="headerThree">
                <button class="accordion-button" type="button" data-bs-toggle="collapse" data-bs-target="#collapseThree" aria-expanded="false" aria-controls="collapseThree">
                    問題 3：檢視AI生成照片的時候，什麼地方容易有缺陷，可以注意哪裡？
                </button>
            </h2>
            <div class="accordion-collapse collapse" id="collapseThree" aria-labelledby="headerThree" data-bs-parent="#accordionExample">
                <div class="accordion-body">
                    解答 3：光影和背景。
                </div>
            </div>
        </div>
    </div>

Carousel 輪播

Carousel (輪播) 常用於建立圖片輪播區 (如首頁 banner )，能自動或手動切換圖片、文字或其它內容。

class 名稱 說明
carousel slide 建立滑動輪播。
carousel-fade 淡入淡出效果。
carousel-item active 輪播項目，第一張一定要有 active。
data-bs-ride="carousel" 自動輪播。
carousel-control-prev/next 左右切換按鈕。
data-bs-interval="3000" 輪播間隔時間，單位是毫秒 (3000 = 3 秒)。
data-bs-pause="hover" 滑鼠移到 Carousel 上時暫停自動輪播。
carousel-indicators 點選小圓點可切換到不同張圖片。
carousel-caption 每張圖片上顯示標題或描述文字。
img-thumbnail 縮圖樣式，帶有邊框與內距。

基本輪播


    <div id="carouselExample1" class="carousel slide carousel-fade mb-3" data-bs-ride="carousel" data-bs-interval="3000" data-bs-pause="hover">
        <div class="carousel-inner">
            <div class="carousel-item active">
                <img src="https://picsum.photos/800/300?random=1" class="d-block w-100" alt="圖片1">
            </div>

    </div>
