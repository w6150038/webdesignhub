排版簡介

Bootstrap佈局主要是使用容器(Container)及網格系統包括行(Rows)與欄(Columns)的組合來進行排版。

斷點(Breakpoint)與容器(Container)

  • 一個頁面中可以有多個容器，容器裡面還可以放入另一個容器(巢狀容器)，容器寬度會隨著斷點而有所不同。
  • 斷點：可以視為響應式變化的分界點。
  • Breakpoint class標記 尺寸
    Extra Small 無 <576px
    Small sm >=576px
    Medium md >=768px
    Large lg >=992px
    X-Large xl >=1200px
    XX-Large xxl >=1400px

Grid System

  • Bootstrap的網格系統(Grid System)使用Flexbox建構，使用容器(Container)、行(Rows)和欄(Columns)的組合進行排版，並隨著螢幕可見區域(Viewport)自動調整。
  • Grid System外層是容器(Container)，容器內使用row樣式定義行，每一行總共有12欄，col樣式是指定跨欄合併的欄數。

CSS-Media Queries (@media)

  • 
                  @media "媒體類型" (媒體特徵) {
    
                   CSS樣式
    
                  }
  • 媒體類型是用來指定適用的類型，當省略媒體類型時預設會使用all。

CSS-Media Queries (@media)

媒體類型 說明
all 所有類型
print 印表機列印
screen 電腦螢幕

範例


              //列印時套用的CSS樣式
                @media print {
                  p {
                    font-size: 10pt;
                  }
                }
            

              //電腦螢幕套用的CSS樣式
                @media screen {
                  p {
                    font-size: 13px;
                  }
                }
            

              //列印、電腦螢幕套用的CSS樣式
                @media screen, print {
                  p {
                     color: green;
                  }
                }
  • 媒體特徵是指符合的條件，必須使用括號()包起來，特徵只要有範圍就可以加上前綴 min-* 或 max-* 查詢最小值或最大值。
  • min-width或min-height是最小值，是指螢幕寬度或高度必須大於此值才符合條件。例如min-width: 576px，表示大於或等於576px都符合。
  • max-width或max-height是最大值，是指螢幕寬度或高度必須小於此值才符合條件。例如max-width: 600px，表示小於或等於600px都符合。
