Navbar 導覽列

class 名稱 說明
navbar 導覽列主要外框。
navbar-expand-lg 導覽列在大螢幕展開，在小螢幕收合。
bg-purple 背景顏色。
container-fluid 導覽列寬度為滿版。
navbar-brand 放置品牌LOGO或網站名稱。
navbar-toggler 在小螢幕上會出現「漢堡選單」，必須與collapse搭配使用。
data-bs-toggle="collapse" 表示點擊這個按鈕 (漢堡 icon) 會觸發「收合或展開」功能。
data-bs-target="#navbarNav" 指定要收合或展開的區塊 id (這裡是 id="navbarNav")。按下按鈕會控制這個區塊。
aria-controls="navbarNav" 無障礙輔助，說明這個按鈕控制的是哪個區塊 (對應 id)。
aria-expanded="false" 表示目前導覽列是否展開，JavaScript會自動根據開關狀態切換 true/false。
aria-label="Toggle navigation" 無障礙輔助，給螢幕閱讀器朗讀的文字。讓使用者知道這是切換導覽列的按鈕。
navbar-toggler-icon 「漢堡選單」的 icon 圖示。
collapse navbar-collapse 將選單區塊折疊起來，點擊漢堡按鈕會展開或收合這個區塊。
navbar-nav 導覽列連結的容器，裡面會放多個nav-item。
nav-item 單一一個導覽列選項或項目。
nav-link 放在<a>超連結標記裡，可以控制超連結樣式 (顏色、間距等)。
aria-current="page" 無障礙輔助，告訴輔具工具這個連結就是目前所在的頁面，通常搭配 class="active"。

                                        </button>

下拉式選單

  • ms-auto：設定「右邊自動外距」，將元素推向左邊。
  • me-auto：設定「左邊自動外距」，將元素推向右邊。
  • mx-auto：設定置中對齊。
  • 將導覽項目靠左，登入/註冊靠右：
    <ul class="navbar-nav me-auto">
     導覽連結靠左
    </ul>
    <form class="d-flex gap-2">
     登入按鈕靠右
    </form>
  • 將導覽項目置中，登入/註冊靠右：
    <ul class="navbar-nav mx-auto">
     導覽連結置中
    </ul>
    <form class="d-flex gap-2">
     登入按鈕靠右
    </form>
  • 將導覽項目靠右，登入/註冊靠右：
    <ul class="navbar-nav ms-auto">
     導覽連結靠右
    </ul>
    <form class="d-flex gap-2">
     登入按鈕靠右
    </form>
  • flex-column、flex-lg-row：小螢幕垂直排列，大螢幕橫向排列。
  • ms-lg-auto：大螢幕時靠右。
  • mt-3、mt-lg-0：小螢幕增加上邊距，大螢幕取消邊距。
  • form-control：讓input更美觀。
  • align-items-lg-center：對齊搜尋框與按鈕。
  • me-lg-2：大螢幕時讓搜尋欄和按鈕之間有距離。
