Modal 彈跳視窗

class 名稱 說明
按鈕的 data-bs-toggle="modal" 告訴 Bootstrap 這是要開啟 modal。
按鈕的 data-bs-target="#exampleModal" 要開啟的目標 modal ID 是 exampleModal。
modal 彈跳視窗的主樣式
fade 讓 Modal 出現時有淡入淡出的動畫效果。
id="exampleModal" 必須與按鈕的 data-bs-target="#exampleModal" 一樣。
tabindex="-1" 按 ESC 鍵可關閉彈跳視窗。
aria-labelledby="exampleModalLabel" 無障礙輔助工具，指出標題是誰。
aria-hidden="true" 預設隱藏，直到被觸發。
modal-dialog modal 的對話盒容器，可以在這裡加上 modal-lg, modal-sm, modal-xl 改變彈跳視窗大小。
modal-content 彈跳視窗裡可看見的內容。
modal-header Modal 頭部區塊。
modal-title 標題文字。
btn-close 右上角的 ✖ 關閉按鈕，
data-bs-dismiss="modal" 配合 btn-close，關閉彈跳視窗。
modal-body 放置彈跳視窗的內容，可放入文字、表單、圖片、清單。
modal-footer Modal 底部區塊。 

    <div class="col-12 col-md-6 col-lg-4 bg-dark">
        <h4 class="text-primary">modal-sm 彈跳小視窗</h4>
        <button type="button" class="btn btn-success" data-bs-toggle="modal" data-bs-target="#modalsm">
            開啟視窗
        </button>
        <div class="modal fade" id="modalsm" tabindex="-1" aria-labelledby="modalsm" aria-hidden="true">
            <div class="modal-dialog modal-sm">
                <div class="modal-content">
                    <div class="modal-header">
                        <h5 class="modal-title text-dark" id="modalsm">Modal 彈跳視窗</h5>
                        <button type="button" class="btn-close" data-bs-dismiss="modal" aria-label="Close"></button>
                    </div>
                    <div class="modal-body text-dark">
                        這是用 Bootstrap 做的 Modal 彈跳小視窗。
                    </div>
                    <div class="modal-footer">
                        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">關閉</button>
                        <button type="button" class="btn btn-success">儲存變更</button>
                    </div>
                </div>
            </div>
        </div>
    </div>
