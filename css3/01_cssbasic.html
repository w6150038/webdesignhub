CSS3介紹

CSS：串接樣式表(Cascading Style Sheet)，簡稱CSS樣式表。

CSS基本格式

  • CSS樣式表是由選擇器(selector)與樣式規則(rule)組成。
  • CSS套用目標可以是HTML標記、class屬性或id屬性。
  • 
								h1, p {
    
								 font-size: 12px;
    
								 line-height: 16px;
    
								 border: 1px #336699 solid;
    
								}
    說明：
    h1, p是選擇器，如果有兩個以上的選擇器，以逗號,隔開。
    font-size: 12px;是樣式規則，font-size是屬性，12px是設定值。一個選擇器裡可以設定多種不同的樣式規則，以分號;隔開；一個屬性裡可以有多個設定值，以空白隔開。
  • CSS樣式註解文字：/* 註解文字 */

宣告CSS

  • 行內宣告(Inline)
    1. 將CSS樣式寫在HTML標記裡。
    2. 語法：
      <h2 style="color: red; font-family: Broadway BT; border: 3px #336699 solid;">行內宣告</h2>
      顯示結果：

      行內宣告

  • 內嵌宣告(Embedding)
    1. 將CSS樣式放在<head></head>標記裡，並且以<style type="text/css">...</style>標記宣告。
    2. 只能套用本身的網頁。
    3. 
										<html>
      
										<head>
      
										 <style type="text/css">
      
										  h2 {
      
										   color: red;
      
										   font-family: Broadway BT;
      
										   font-weight: bold;
      
										  }
      
										  h3 {
      
										   color: #0000CC;
      
										   font-family: ParkAvenue BT;
      
										   font-weight: bold;
      
										  }
      
										 </style>
      
										</head>
      
										<body>
      
										  <h2>內嵌宣告h2標記</h2>
      
										  <h3>內嵌宣告h3標記</h3>
      
										</body>
      
										</html>
      內嵌宣告
  • 連結外部樣式檔(Linking)
    1. 直接用編輯器寫CSS樣式規則，可省略<style></style>標記，並存檔成.css檔名。
    2. 在head標記裡呼叫.css樣式檔。
    3. 語法：<link rel="stylesheet" type="text/css" href="檔名.css">
    4. 當一個HTML文件同時套用了三種宣告方式，並有重複樣式時，優先順序為「行內宣告」>「內嵌宣告」>「連結外部樣式檔」。
    5. 連結外部樣式檔除了連結本地伺服器.css檔，也可以連結CDN節點。
    6. Bootstrap官網可查詢到CDN連結，只要在href帶入CDN連結網址即可。
    7. CDN：內容傳遞網路(Content Delivery Network)，是一種在網路上傳輸的快取機制，透過分散在全球不同節點的CDN服務，能夠達到有效分流，加快資源傳輸的速度，讓使用者可以就近取得所需的資源。

CSS選擇器

  • 標記名稱選擇器(Element)：
    使用此選擇器可以將html文件裡「所有相同的標記」都套用同一種樣式。
    div {font-size: 16px; color: #FFFFFF;}
    上述語法表示將HTML文件裡「所有的div標記」都套用到{}內的樣式。
  • 通用選擇器(*)：
    * {font-size: 16px; color: #FF0000;}
    上述語法表示將HTML文件裡「所有的標記」都套用到{}內的樣式。
  • class選擇器：
    1. 在HTML標記加入class屬性，屬性名稱不要使用HTML標記名稱。
      語法：<標記名稱 class="class名稱">
      例如：<font class="txt">
    2. 在CSS樣式裡加入class選擇器。
      語法：.class屬性名稱{樣式規則1; 樣式規則2;...}
      例如：.txt {font-size: 16px; color: #FFFFFF; font-weight: bold;}
    3. 如果有多個標記名稱指定相同class名稱，但只希望在某一個標記裡套用class選擇器的樣式，則可以在class選擇器前加上標記名稱。
      語法：標記名稱.class屬性名稱{樣式規則1; 樣式規則2;...}
      例如：font.txt {font-size: 16px; color: #FFFFFF; font-weight: bold;}
  • id選擇器：
    1. HTML標記裡的id屬性名稱是用來辯識元件用的，就像身份證一樣，通常是配合由JavaScript語法來控制HTML元件時，利用id名稱來取用HTML元件屬性，因此id名稱在同一份HTML文件裡只能是唯一的，不能重複。
    2. 不要使用HTML標記當作id名稱。
    3. 語法：#id屬性名稱{樣式規則1; 樣式規則2;...}
      
										  <html>
      
										  <head>
      
										  <title>使用id選擇器</title>
      
										  <style type="text/css">
      
										   #font_style {
      
										    font-size: 24px;
      
										    color: red;
      
										    font-family: Broadway BT;
      
										    font-weight: bold;
      
										    border: 1px #336699 solid;
      
										   }
      
										  </style>
      
										  </head>
      
										  <body>
      
										   <font id="font_style">使用id選擇器</font>
      
										  </body>
      
										  </html>
      id選擇器
  • 屬性選擇器：
  • 用來篩選HTML標記裡的屬性。
  • 語法：HTML標記名稱[HTML標記屬性]{CSS樣式規則1; CSS樣式規則2;...}
    例如：指定超連結標記<a>的背景顏色為藍色，但只套用在有target屬性的元件。
    語法：a[target="設定值"]{background-color: blue;}
  • 屬性選擇器的篩選方式有6種：
    [attribute="value"] 屬性值等於value
    [attribute~="value"] 屬性值包含完整value
    [attribute|="value"] 屬性值等於value或以value-開頭
    [attribute^="value"] 屬性值開頭有value
    [attribute$="value"] 屬性值結尾有value
    [attribute*="value"] 屬性值裡面有出現value
  • 反向選擇：
    如果整份HTML文件中除了<p>標記以外，其他標記都要套用同一種樣式，就可以使用反向選擇。
    :not(p){color: red;}
  • 虛擬類別選擇器(偽選擇器)(:)：
    1. 連結虛擬類別：依據超連結元素的狀態來選取物件。
      選擇器 說明
      :link 未訪問過的連結
      :visited 已訪問過的連結
      :hover 滑鼠游標碰到元素
      :active 被按下滑鼠左鍵的元素
      :focus 取得焦點的元素
    2. 結構性虛擬類別：依照元素的特性來選擇特定子元素。
      選擇器 說明
      p:first-child 找到p兄弟元素的第1個元素(所有元素)
      p:last-child 找到p兄弟元素的最後第1個元(所有元素)
      p:nth-child(n)
      p:nth-child(odd)       		(n)找第n個元素
      (odd)找奇數元素
      p:nth-child(even) (even)找偶數元素
      找所有元素
      p:first-of-type 找到p兄弟元素的第1個元素(只找同類元素)
      p:last-of-type 找到p兄弟元素的最後第1個元(只找同類元素)
      p:nth-of-type(n) (n)找第n個元素，同樣可輸入odd或even找奇偶數元素(只找同類元素)
    3. UI(User Interface使用者介面)狀態虛擬類別：網頁元素在瀏覽器呈現的狀態。
      選擇器 說明
      E:enabled UI啟用的元素
      E:disabled UI禁用的元素
      E:checked UI被選取的元素
    4. 目標虛擬類別
      選擇器 說明
      :target 錨點的目標元素
  • 當CSS選擇器的樣式設定有衝突時，會採用優先權高的選擇器模式。
    ID選擇器 > Class選擇器 > Element標記名稱選擇器 > *通用選撫器
  • 屬性選擇器、偽選擇器與class選擇器權重相同，當選擇器權重相同時，後者會覆蓋前者，但是當選擇器的設定值有宣告「!important」時，就會以此設定為優先。
    
								<style type="text/css">
    
								 .ex1 {color: blue !important;}
    
								 .ex1 {color: red;}
    
								</style>
    照理說，後者會覆蓋前者，顏色應該是紅色，但前者宣告了「!important」，具有更高的優先權，所以顏色為藍色。

CSS度量單位

  • 絕對單位：
    1. pixels(px)、point(pt)、公分(cm)、英寸(in)、公釐(mm)。
    2. 不會隨著外層物件的單位變動，譬如12px就會固定以12px呈現。
    3. pixcel(px)是螢幕上的一個點，會與螢幕解析度有絕對關係。
  • 相對單位：
    1. 倍數(em)、倍數(rem)、百分比(%)。
    2. 會隨著外層物件的單位變動，
    3. 
										<style type="text/css">
      
										 body {
      
										  font-size: 12px;
      
										 }
      
										 h1 {
      
										  font-size: 2em; //body裡font-size: 12px的2倍，所以是24px。
      
										 }
      
										</style>
  • 一般瀏覽器預設的font-size是16px，如果上層元件沒有設定單位，則2em就會是16px*2=32px。
  • 倍數(rem)只受html影響，
  • 
								<style type="text/css">
    
								 html {font-size: 30px;}
    
								 div {
    
								  font-size: 60px; //此設定值單位為px，所以不受html影響。
    
								 }
    
								 h1 {
    
								  font-size: 2rem; //h1的font-size為html裡font-size: 30px的2倍，所以是60px。
    
								 }
    
								</style>

CSS顏色表示法

CSS常用顏色表示法

語法 範例 說明
{color: 顏色名稱} {color: blue} 以顏色名稱表示
{color: #16進位色彩} {color: #6600CC} 16進位色彩(HEX color)
{color: rgba(R.G.B.a)} {color: rgba(255.0.0.3)} RGBA色彩

CSS常見顏色名稱

black(黑色) blue(藍色) gray(灰色) green(綠色) olive(橄欖色)
purple(紫色) red(紅色) silver(銀色) white(白色) yellow(黃色)
  • 16進位色彩(HEX色彩)：由「#」加上6個數字表示，前兩碼代表RGB色彩中的R，中間兩碼代表RGB色彩中的G，後兩碼代表RGB色彩中的B。
  • 最小值是0，最大值是F。
  • RGBA色彩(Red-Green-Blue-Alpha)：RGB三個顏色的強度為0到255之間的整數，或0%到100%；Alpha是指顏色的不透明度，0是完全透明，1是完全不透明。
