CSS盒子模型(Box Model)

HTML的區塊元素都可以當作一個盒子，CSS針對盒子的位置與外觀定義了盒子模型，包括邊距(margin)、邊框(border)、填充(padding)。

content

上面的div元件使用CSS設定寬度150px，但實際上佔版面的總寬度與總高度如下：
總寬度=左邊距+左邊框+左填充+寬度+右填充+右邊框+右邊距
總高度=上邊距+上邊框+上填充+高度+下填充+下邊框+下邊距

box-sizing

  • box-sizing屬性是定義元件的寬度和高度是否包括填充(padding)與邊框(border)。
  • box-sizing: content-box：此為預設值，寬度和高度只包括內容(content)本身的寬度和高度，不包括邊框(border)、填充(padding)和邊距(margin)。
  • div {
       box-sizing: content-box;
       width: 200px;
       border: 20px solid #ffcc00;
       padding: 15px;
       background-color: lightblue;
     }

  • 上面語法在div元件設定了box-sizing: content-box，因此content的寬度就是200px，不會受到border、padding和margin設定的影響。

    content寬度200px

  • box-sizing: border-box：此設定的寬度和高度包括了內容(content)、填充(padding)、和邊框(border)，但不包括邊距(margin)。
  • div {
       box-sizing: border-box;
       width: 200px;
       border: 20px solid #ffcc00;
       padding: 15px;
       background-color: lightblue;
     }

  • 上面語法在div元件設定了box-sizing: border-box，此設定涵蓋了border: 20px、padding: 15px，因此content的寬度就只剩下130px。

    content寬度剩下130px

Flexbox彈性盒子模型(Flexible Box Model)

  • 在盒子加入主軸(main axis)與交叉軸(cross axis)，讓盒子能夠彈性縮放。當一個HTML元件利用CSS將display設定為flex，該元件就成為一個彈性容器(flex container)。
  • 彈性盒子(flexbox)是由外層容器(flex container)與內層子元素(flex item)所組合而成。
  • flex container

    flex item 2
  • flexbox靠主軸(main axis)與交叉軸(cross axis)進行子元素的分配與排列，彈性容器(flex container)裡的子元素稱為彈性元素(flex item)，彈性元素會沿著主軸或交叉軸，從起點往終點方向排列。
  • 要產生一個flexbox，只要將HTML元件裡的CSS display屬性設定為flex或inline-flex即可。

flexbox屬性能夠設置flex物件排列的方向與對齊的方式。

flex container屬性

  • 水平對齊text-align與垂直對齊vertical-align不適用於flex，flex是靠主軸(等於text-align)和交叉軸(等於vertical-align)的方式來對齊。
  • flexbox的外層容器(flex container)的屬性主要是設定子元素(item)排列的方向及對齊。
  • 屬性 說明 用法
    display 設置flex排列方式
    • flex：區塊元素
    • inline-flex：行內元素
    		 display: flex;
    display: inline-flex;
    flex-direction 彈性元素的排列方向
    • row：預設值，從左到右排列
    • row-reverse：從右到左排列
    • column：從上到下排列
    • column-reverse：從下到上排列
    		 flex-direction: row;
    flex-direction: row-reverse;
    flex-direction: column;
    flex-direction: column-reverse;
    flex-wrap 設置flex容器單行或多行
    • nowrap：不斷行，維持單行
    • wrap：多行
    • wrap-reverse：多行，且與wrap起點與終點相反
    		 flex-wrap: nowrap;
    flex-wrap: wrap;
    flex-wrap: wrap-reverse;
    flex-flow flex-direction與flex-wrap綜合應用 flex-flow: row nowrap;
    justify-content 設置彈性元素在主軸的對齊方式
    • flex-start：靠主軸起點對齊
    • flex-end：靠主軸終點對齊
    • center：置中對齊
    • space-between：平均分配寬度，左右兩個元素貼齊邊緣
    • space-around：平均分配寬度和間距
    		 justify-content: flex-start;
    justify-content: flex-end;
    justify-content: center;
    justify-content: space-between;
    justify-content: space-around;
    align-items 設置彈性元素在行內交叉軸的對齊方式
    如果彈性元素設定align-self屬性，則align-items無效
    • flex-start：靠交叉軸起點對齊
    • flex-end：靠交叉軸終點對齊
    • center：置中對齊
    • baseline：彈性元素內容基線對齊
    • stretch：將彈性元素拉伸對齊交叉軸起點與終點
    		 align-items: flex-start;
    align-items: flex-end;
    align-items: center;
    align-items: baseline;
    align-items: stretch;
    align-content 設置彈性元素在整體交叉軸的對齊方式
    • flex-start：靠交叉軸起點對齊
    • flex-end：靠交叉軸終點對齊
    • center：置中對齊
    • space-between：平均分配高度，頂部和底部兩個元素貼齊邊緣
    • space-around：平均分配高度和間距
    • stretch：拉伸至整個空間
    		 align-content: flex-start;
    align-content: flex-end;
    align-content: center;
    align-content: space-between;
    align-content: space-around;
    align-content: stretch;

flex items屬性

屬性 說明 用法
align-self 調整單一子元素在交叉軸的位置，屬性值與align-items相同。 align-self: flex-start;
align-self: flex-end;
align-self: center;
align-self: baseline;
align-self: stretch;
flex-basis 設定初始寬度與高度。
若flex-direction: row，則flex-basis決定width。
若flex-direction: column，則flex-basis決定height。
設定值：
  • auto：根據自身尺寸縮放(預設值)。
  • width/height：數值或百分比。
 flex-basis: 80px;
flex-basis: auto;
flex-basis: 10em;
flex-grow 分配彈性盒子剩餘空間的伸展權重，設定值為大於或等於0的數值，預設值為0。 flex-grow: 3;
flex-grow: 0.6;
flex-shrink 分配彈性盒子不足空間的壓縮權重，設定值為大於或等於0的數值，預設值為1。 flex-shrink: 2;
flex-shrink: 0.6;
flex 同時設置flex-grow、flex-shrink、flex-basis。
  • flex-grow：預設值為0。
  • flex-shrink：預設值為1。
  • flex-basis：預設值為auto。
 flex: 10em;
flex: 1 30px;
flex: 2 2 10%;
order 設定flex items排列順序，數值為整數，可以是負數，預設值為0。 order: 2;

  • 分配彈性盒子剩餘空間的伸展權重。
  • 如果彈性盒子裡所有子項目都設定相同的flex-grow，則會平均分配剩餘空間。

flex綜合應用

  • 順序為flex-grow、flex-shrink、flex-basis。
  • 預設值為0 1 auto，也就是當container有剩餘空間時不伸展，空間不足時會壓縮，尺寸會隨著內容縮放。
  • CSS語法 說明
    flex: auto; 相當於flex: 1 1 auto;
    flex: none; 相當於flex: 0 0 auto;
    flex: 2; 1個整數值，表示flex-grow，相當於flex: 1 1 0%;
    flex: 50px; 1個帶單位數值，表示flex-basis。
    flex: 1 50px; 1個整數值及1個帶單位數值，表示flex-grow與flex-basis。
    flex: 1 1; 2個整數值，表示flex-grow與flex-shrink。
    flex: 1 1 50px; 3個值，表示flex-grow、flex-shrink與flex-basis。
