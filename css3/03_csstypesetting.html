邊界margin

  • 在邊框(border)外圍，用來設定元件的邊緣距離，共有上下左右四邊屬性可設定。
  • margin-top：上邊界。
    margin-right： 右邊界。
    margin-bottom：下邊界。
    margin-left：左邊界。
    口訣：上右下左。
  • 長度單位：pt、px、%、auto(預設值)。
  • 語法1：選擇器{margin-top:設定值;}
    語法2：
      div {
       margin-top: 20px;
       margin-right: 40pt;
       margin-bottom: 100%
       margin-left: auto;
      }
    語法3：margin:上邊界值 右邊界值 下邊界值 左邊界值(以空白隔開)
      div {margin: 5px 10px 15px 20px} /*上：5px，右：10px，下：15px，左：20px*/
      div {margin: 5px;} /*四邊界皆為5px*/
      div {margin: 5px 10px;} /*上：5px，右：10px，下：5px，左：10px*/
      div {margin: 5px 10px 15px;} /*上：5px，右：10px，下：15px，左：10px*/
  • 為了方便控制網頁中元件的位置，利用<div>標記設好網頁內容寬度，並且水平置中，將margin屬性上下設為0，左右就根據瀏覽器大小自行調整。
    語法：margin: 0 auto;

border-image花樣邊框

  • 語法：選擇器{border-image:source slice width repeat;}
  • source：url(圖片路徑)，必填。
  • slice：切出圖片使用的界線，必填。
  • width：圖片寬度，可省略。
  • repeat：圖片填充方式，可省略。
    1. stretch：把圖片拉伸至整個邊框區域。
    2. repeat：重複填滿。
    3. round：重複填滿並自動調整圖片大小。

邊界留白padding

  • 指邊框(border)內與HTML元件邊緣的距離。
  • 屬性 屬性名稱 設定值
    padding-top 上邊界留白距離 px, pt, %, auto
    padding-right 右邊界留白距離
    padding-bottom 下邊界留白距離
    padding-left 左邊界留白距離
  • 語法：選擇器{padding:上邊界留白 右邊界留白 下邊界留白 左邊界留白;}
  • 邊界margin、邊框border與邊界留白padding通常都是搭配使用的。

網頁元件定位position

  • 一般定位
    • 屬性 屬性名稱 設定值
      position 設定元件位置排列方式 absolute(絕對定位)
      relative(相對定位)
      static(靜止)
      width、height 指定元件寬度與高度 px, pt
      left 指定元件與左邊界距離(x座標) px, pt
      top 指定元件與上邊界距離(y座標) px, pt
      overflow 元件超出邊界的顯示方式 visible
      hidden
      scroll
      auto
      visibility 是否顯示 visible
      hidden
      inherit
    • position
      • 通常會與<div>標記搭配使用，是用來將元件精確定位，top、right、bottom、left屬性決定元件的位置。
      • static(靜態定位)：元件依照常規流(Normal Flow)，此時top、right、bottom、left及z-index均無效。
      • absolute(絕對定位)：元件脫離常規流，設定top、right、bottom、left時，元件原本所在的位置會移除，下一個元件會順著常規流移到原本元件的位置，並且以有使用position定位的上一層元件(父元件)的左上角點為原點來定位，如果找不到父元件，則以<body>左上角點為原點來定位。
      • relative(相對定位)：元件依照常規流，設定top、right、bottom、left時，原本的位置會保留下來，下一個元件會固定不動，並以元件本身的左上角為原點來定位。
      • absolute與relative
      • fixed(固定定位)：元件脫離常規流，直接指定元件在viewport(裝置寬跟高的固定位置)，就算拉動捲軸也不會改變元件在viewport的位置。
      • sticky(粘滯定位)：必須指定top、right、bottom、left才能發揮作用，根據使用者捲動的行為來定位，一般狀況下是相對定位，但當元件超出目標區域時就變成固定定位。IE/Edge15不支援，Safari需要加上「-webkit-」。
    • overflow
      • visible：不管元件長寬，內容完全呈現。
      • hidden：超過長寬的內容就不會顯示。
      • scroll：無論內容有無超出長寬，都加入捲軸。
      • auto：視情況是否顯示捲軸。
      • overflow
    • float
      • 性質 性質名稱 說明
        float: left;
        float: right;         		允許文字與圖片排列 left：圖片在左側
        right：圖片在右側
        clear: left;
        clear: right;
        clear: both;         		清除float設定 left：清除float: left設定
        right：清除float: right設定
        both：清除float: left與float: right設定
      • float
  • 圖層定位z-index
    • 必須與position屬性一起使用。
    • 當元件相互重疊時，可以指定元件之間的上下層次順序。
    • z-index數值大的元件會排在數值小的元件上面，也就是數值越大，層次越高。
    • 如果兩個元件的z-index值相同，則會以程式後寫者位於上方(back-to-front)。
    • z-index
  • calc()函式
    • 可以使用在任何一個需要數值的地方，例如px、length、angle、time、number，單位：px、％、em、rem。
    • 語法：calc{expression}
      例如：width: calc(100% - 80px)
    • +-*/運算符號前後必須空一格。
    • calc(100% - 400px)

超連結特效

  • 超連結有4種狀態：尚未連結(link)、已連結(visited)、滑鼠移到連結時(hover)、執行中(active)。
  • 語法：
      a {樣式屬性;屬性值;}
      a:link {樣式屬性;屬性值;}
      a:visited {樣式屬性;屬性值;}
      a:active {樣式屬性;屬性值;}
      a:hover {樣式屬性;屬性值;}
  • 通常使用a選擇器與a:hover選擇器就可以做出效果。
  • 圖片超連結特效：a:hover {background-image: url(圖片來源);}。

滑鼠游標特效cursor

語法1：cursor {滑鼠游標樣式;}
語法2：body {cursor: url(游標檔.cur);}

display區塊模式

display: block 區塊固定大小，當文字超過區塊時，文字會換行。
display: inline 區塊隨著內容變動，當文字超過區塊時，區塊會擴大。
